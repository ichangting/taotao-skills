# 安装指南（给一线教师）

本仓库的技能（Skill）是给 **WorkBuddy**（以及兼容的 Claude Code / CodeBuddy 类 AI 客户端）
用的"教学小助手"。下面教你怎么装、怎么用。全程不需要写代码。

---

## 一、装到哪里？

客户端识别技能靠**文件夹位置**，二选一：

| 位置 | 作用范围 | 路径 |
|---|---|---|
| **用户级** | 你所有项目都能用（推荐） | `C:/Users/你的用户名/.workbuddy/skills/` |
| **项目级** | 仅某一个项目里能用 | `你的项目文件夹/.workbuddy/skills/` |

> 不知道用户名？打开资源管理器地址栏输入 `%USERPROFILE%` 回车即可看到。

---

## 二、安装一个技能（3 步）

以"语文备课"为例：

1. 从本仓库把 `skills/k12-lesson-planning-zh` 这个**整个文件夹**下载下来
   （点仓库里这个文件夹 → 右上角 `…` / Download，或用 Git）。
2. 把它**整个**复制到上面的 `skills/` 目录里，结果应是：
   `…/.workbuddy/skills/k12-lesson-planning-zh/skill.md`
3. 重启一下客户端（让它在 `skills/` 里"发现"新技能）。

其余技能（王力视角、文档审查、词频分析……）同理，各复制各的文件夹即可。

---

## 三、语文备课技能要多装一步环境

只有 `k12-lesson-planning-zh` 在"生成 Word 文档"时需要本机有 Python。
别的技能是纯提示词，复制过去就能用，跳过这节。

1. 安装 Python（官网 python.org，勾选 "Add to PATH"）。
2. 打开命令行，装一个库：

```bash
pip install python-docx==1.1.2
```

3. 验证：命令行输入 `python` 能进得去、能 `import docx` 不报错，就成功了。

---

## 四、怎么确认装好了？

装完后，在客户端里直接说一句话测试：

> 「帮我备七上语文《春》」

- 若它开始问你年级 / 文体、或先给你一份速览草稿 → **装好了**。
- 若它完全没反应、或说"没有这个技能" → 检查文件夹位置对不对、
  文件夹名是不是 `k12-lesson-planning-zh`、里面有没有 `skill.md`。

---

## 五、常见问题

**Q：复制过去后客户端没识别？**
A：确认路径是 `…/skills/<技能名>/skill.md`，`<技能名>` 必须和仓库里一致；
   然后重启客户端。

**Q：备课技能说"找不到 python-docx"？**
A：回到第三节，确认 pip 装的是同一个 Python（有时电脑装了多个 Python，
   用 `python -m pip install python-docx` 更稳）。

**Q：生成的是 Word，但中文显示成方块 / 乱码？**
A：用仓库里的 `fix_fonts.py` 跑一下即可（详见该技能 README 第四节）。

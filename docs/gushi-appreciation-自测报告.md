## 三、修正后鉴赏卡（clean，可直接用于备课）

```
【古诗文鉴赏卡】天净沙·秋思 · 七年级上册 · 曲

一、原文与出处
- 原文：枯藤老树昏鸦，小桥流水人家，古道西风瘦马。夕阳西下，断肠人在天涯。
- 作者：马致远（元）  出处：《全元散曲》（统编教材据中华书局本）
- 〔经 doc-review 古文专项收口：作者/篇目/原文无误〕

二、主旨情感
- 主流定评：抒发漂泊天涯的游子深秋思乡、孤寂愁苦之情。
- 争议提示：学界基本一致；可讨论"断肠"是直写乡愁还是兼含仕途失意之悲。

三、意象分析
- 昏鸦（原句"枯藤老树昏鸦"）：常规含义 日暮归巢之鸦，本篇以"昏"点明暮色，反衬游子犹未归。
- 瘦马（原句"古道西风瘦马"）：常规含义 羁旅之马，本篇指游子长途困顿之态。
- 夕阳（原句"夕阳西下"）：常规含义 迟暮/思归，本篇烘托断肠之境。

四、艺术手法
- 白描：前三句纯名词意象并列（枯藤/老树/昏鸦…），不着形容词而画面自显。
- 借景抒情：全篇无一"愁"字，景语皆情语，以萧瑟秋景寓断肠之愁。
- 动静/反衬："小桥流水人家"的温馨静景，反衬"古道西风瘦马"的漂泊孤苦。

五、名句赏析
- 原句："夕阳西下，断肠人在天涯"
  · 字面：夕阳沉落，漂泊的游子困于天涯尽头。
  · 手法：以景结情 + 反衬（前文人家温馨 vs 自身天涯）。
  · 情感：将羁旅之愁推至顶点，"断肠"二字凝练沉痛。

六、背诵默写易错字（回查 guwen-cidian）
- 涯：正写"涯"（三点水+厓），易误写"崖"（山崖）；非通假。
- 昏：常字（日+氏），非通假，意为日暮。
- 鸦：正写"鸦"，易误"鸭"。

七、中考链接（未安装考情技能，基于通用经验）
- 常见题型：意象作用题（"昏鸦/瘦马"作用）、情感主旨题、以乐景写哀情手法题。
- 提示：以当年江西/全国中考真题为准。

八、定评 vs 商榷
- 主流定评：羁旅思乡之愁，白描+借景抒情为其艺术标识。
- 可讨论点：①"断肠人"是否即马致远自况（可结合其一生漂泊展开）；②"小桥流水人家"的温馨画面是单纯景物还是暗含"他人团圆、唯我孤旅"的对照张力。
```

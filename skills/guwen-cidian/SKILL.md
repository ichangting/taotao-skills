# 初中文言词典（guwen-cidian）

## 目的

帮中学生和一线教师**速查**文言字词——不是堆释义，而是给 actionable释义："这句取哪个义、为什么、中考考不考、教材用的哪个版本"。覆盖五大场景：① 单字/词查义与一词多义辨析；② 通假字/古今异义/词类活用提示；③ 句子/课文逐字串讲；④ 江西中考 104 高频文言词速查与自测；⑤ 统编教材版本核对。

源自原 `wordbank-reviewer` 技能，但**彻底剥离**了"私有题库 JSON 审查/lint"那套与公开受众无关的构建流水线，把真正有价值的中考文言词数据沉淀为本技能的核心知识库；后又并入原 `doc-review` 的古诗文基础笔记与教材版本数据，现为**完全自包含**的文言文一站式技能（不再跨技能依赖 `doc-review`）。

## 何时使用

- "之/以/而 这些字在文言文里是什么意思？""这个词在句里该取哪个义？"
- "这句古文怎么翻译""帮我把这段文言文串讲一下"
- "通假字有哪些""这个词是古今异义吗""这里是不是词类活用"
- "江西中考文言文常考哪些词""帮我自测一下高频词"
- "《桃花源记》里'要'字教材为什么注 yāo 不写通'邀'？""这篇古文用的哪个版本？"
- 备课、出题、答疑、学生自学任何涉及文言字词的场景。

## 核心能力（五种模式，按需切换）

### 模式一：单字 / 词查义 + 一词多义辨析

**流程**：
1. 在 `references/文言词表/` 中定位该字：
   - 先看 `江西省中考文言文高频词表.md`（104 个江西中考真题高频词，含义项 + 标注年份的真题例句）——命中即优先用此表，因为例句来自真实中考卷。
   - 未命中再查 `000中学文言文常用词常用义-合集.md`（775 词总索引，按出现频次五星排序）。
   - 更细的义项整理见 `文言文高频词义项整理_第3组.md`、`江西省中考真题文言文高频词义项整理_第2组.md`。
   - 双音节词见 `003古汉语常见双音节词.md`。
2. **若用户只问"什么意思"**：按中考/教材频度列出全部常见义项，每条配例。
3. **若用户给了具体句子（一词多义辨析）**：必须做"语境判定"——从这字的多个义项中**选定本句正确义项**，并写明判定依据（看句子成分、搭配、是否主谓之间等），不要只罗列义项。

**输出格式**：按 `references/字词卡片模板.md` 输出锁死格式的字词卡片。

**完成标准**：查到的字与用户输入完全一致（含异体字核对）；义项列表非空且按中考频度排序；每条义项配有例句（优先中考真题）；若用户给了句子，必须有"本句语境判定"栏；"中考频度"栏有具体数据。

### 模式二：通假字 / 古今异义 / 词类活用 / 虚词实词 提示

本技能**内置**这些专项笔记（合并自原 `doc-review`），无需跨技能引用：

- 通假字 → `references/古诗文基础/初中文言文通假字.md`
- 词类活用 → `references/古诗文基础/初中文言文词类活用.md`、`文言文词类活用.md`
- 古今异义 / 一词多义 → `references/古诗文基础/一词多义与古今异义.md`
- 文言虚词 → `references/古诗文基础/初中常用文言虚词.md`、`初中文言文重点虚词.md`、`文言文常用虚词.md`
- 文言实词 → `references/古诗文基础/文言文实词.md`
- 特殊句式 / 句读 → `references/古诗文基础/文言文特殊句式.md`、`文言句读.md`
- 文化常识 / 默写易错字 → `references/古诗文基础/古代文化常识.md`、`中考语文古诗文默写易错字.md`

**用法**：当用户问到某字是否通假 / 古今异义 / 活用，先查上述文件给出准确说法；并在字词卡片的"易错提醒"或串讲的"特殊用法"栏标注。

**完成标准**：已查阅与用户问题相关的全部专项笔记文件；给出的说法与文件记录一致，无臆造；对不确定或争议内容注明"以教材/官方发布为准"；标注位置（易错提醒/特殊用法栏）正确。

### 模式三：句子 / 课文逐字串讲

**流程**：
1. **逐字解析**：逐字/逐词切分，每个有解释价值的字调用模式一（优先高频词表/合集）。
2. 标出特殊用法：通假、古今异义、词类活用、重点虚词（查模式二的文件）。
3. 给出整句通顺翻译（现代汉语句式，不硬译）。

**输出格式**：按 `references/串讲卡模板.md` 输出锁死格式的串讲卡。

**完成标准**：原句中每个有解释价值的字都有逐词标注；特殊用法（通假/活用/古今异义）已标出或明确写"无"；整句翻译为通顺现代汉语；标注释义优先回查高频词表/合集确保准确。

### 模式四：江西中考 104 高频词 速查与自测

- **速查**：直接基于 `references/文言词表/江西省中考文言文高频词表.md`（104 词，2014–2025 真题例句），按词频或音序列出，标注每个词的中考出现次数与年份分布。
- **自测**：从高频词表中抽 N 个词，给出"请写出其在句中的含义 + 所属义项"的题，配答案（答案回查该表，保证与真题例句一致）。可限定年级 / 年份 / 实词虚词。

**完成标准**：速查结果与高频词表数据一致（词/义项/例句/年份）；自测题目答案回查高频词表验证正确；若用户限定了年级/年份/词性范围，筛选条件已正确应用。

### 模式五：统编教材版本核对

- 当用户问"某篇古文教材用的哪个版本 / 某字教材为什么这么注"，查 `references/教材版本/统编教材版本对照表.md` 锁定该篇目所属年级与所据校注本。
- 遇异文 / 通假 / 注音争议，查 `references/教材版本/版本差异与选用规则.md`（P0-P3 优先级 + 常见版本差异对照表 + 权威出版社）。
- **判定原则**：教学用途优先采用统编教材版本（即使与其他权威本有异）；非教材篇目优先中华书局 / 上海古籍校注本。

**完成标准**：已锁定篇目所属年级与所据校注本（查对照表）；遇争议时给出 P0-P3 优先级判定并说明理由；非教材篇目明确标注推荐版本来源。

## 诚信与合规护栏

- **以权威为准**：释义基线来自统编教材与江西省中考真题。若遇争议或存疑释义，注明"以教材 / 江西省教育考试院当年发布为准"，不臆造。
- **真题例句须忠实**：引用中考例句时保留原句与年份标注，不篡改；本技能数据按 [数据年度更新SOP](../../docs/数据年度更新SOP.md) 每年核对。
- **不冒用学者署名**：本技能是工具性查词，不涉及王力/吕叔湘等学者视角模拟；如结合 `wangli-perspective` 等视角技能产出内容，对外发表须注明为"视角模拟、仅供参考"。
- **品牌声音**：对外呈现的讲解语气参考 [品牌声音与品牌规范](../../docs/品牌声音与品牌规范.md)（准确、干净、不浮夸）。

## 新增词目规范

如需补充词目，套用 `references/文言词表/词条标准模板.md`，保证 `字 / 音 / 词性 / 义项 / 例句(含出处年份) / 频度` 字段齐全，并与高频词表、合集的既有写法一致。

## 参考资料（references/）

本技能完全自包含，references 下分三类：
- `references/文言词表/`：核心文言词知识库（9 份，见上表）。
- `references/古诗文基础/`：通假字 / 词类活用 / 古今异义 / 虚词 / 实词 / 特殊句式 / 句读 / 文化常识 / 默写易错字等结构化笔记（14 篇）。
- `references/教材版本/`：统编教材版本对照表 + 版本差异与选用规则（2 篇）。

> 词目、字形、义项须与基线一致；新增词目请套用 `词条标准模板`。

# 教材版本差异与选用规则

> 本文件供「初中文言词典」在判定通假字、异文、注音时锁定权威依据；亦供 `doc-review` 做"古文引用审核"时回查。
> 核心原则：**教学用途优先采用统编教材版本**；非教材篇目优先采用中华书局 / 上海古籍出版社权威校注本。

## 版本选择优先级

| 优先级 | 版本类型 | 说明 |
|--------|----------|------|
| **P0** | 统编教材版本 | 义务教育/高中统编语文教科书采用版本（最高优先级） |
| **P1** | 权威校注本 | 中华书局、上海古籍出版社等权威出版社校注本 |
| **P2** | 通行善本 | 学术界公认善本（如四部丛刊、四库全书等） |
| **P3** | 其他版本 | 需注明版本来源，供用户参考选择 |

## 特别提示

1. **教学用途优先采用统编教材版本** —— 即使与其他权威版本有差异，也应以教材为准。
2. **通假字保留原则** —— 教材通常保留通假字原貌（如"说"不改为"悦"）。
3. **异体字处理** —— 教材一般采用规范汉字，异体字需转换。
4. **标点差异** —— 古籍原无标点，不同版本断句可能有差异，以教材为准。
5. **非教材篇目** —— 优先采用中华书局、上海古籍出版社权威校注本。
6. **版本注明** —— 学术用途需注明所据版本（如"据中华书局 2020 年版《论语译注》"）。

## 快速查询工具

**统编教材版本查询**：
- 义务教育：人民教育出版社官网 → 电子教材
- 高中：人民教育出版社官网 → 普通高中教科书

**权威数据库**：
- 中国基本古籍库（爱如生）
- 中华经典古籍库（中华书局）
- 国学大师网（免费，但需核对版本）

# 字词卡片模板

> 模式一（单字/词查义 + 一词多义辨析）的锁死输出格式。

```
### 【字词卡片】<字>
- **读音 / 字形**：<音> （如有通假，注明"通 X"）
- **词性**：<实词/虚词/助词/代词…>
- **核心义项（按中考频度排序）**：
  1. <义项1> — 例：<例句>（<出处/年份>）
  2. <义项2> — 例：<例句>
  …
- **本句语境判定**：<若给了句子>本句取第 N 个义项，理由：<看 XX 结构/搭配>
- **中考频度**：<★★★★★ / 出现 N 次 / 江西 2014–2025>
- **易错提醒**：<与哪义易混、辨析窍门>
```

## 完成标准

- [ ] 查到的字与用户输入完全一致（含异体字核对）
- [ ] 义项列表非空，且按中考频度排序
- [ ] 每条义项配有例句（优先中考真题例句）
- [ ] 若用户给了句子，有"本句语境判定"栏
- [ ] "中考频度"栏有具体数据（★等级或出现次数）

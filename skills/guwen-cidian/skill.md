# 初中文言词典（guwen-cidian）

## 目的

帮中学生和一线教师**快速、可靠地查清文言文里的字词问题**——不是堆释义，而是给"在当下这句里到底该取哪个义、为什么、中考考不考"的可操作答案。覆盖四大场景：① 单字/词查义与一词多义辨析；② 通假字/古今异义/词类活用提示；③ 句子/课文逐字串讲；④ 江西中考 104 高频文言词速查与自测。

源自原 `wordbank-reviewer` 技能，但**彻底剥离**了"私有题库 JSON 审查/lint"那套与公开受众无关的构建流水线，把真正有价值的中考文言词数据沉淀为本技能的核心知识库。

## 何时使用

- "之/以/而 这些字在文言文里是什么意思？""这个词在句里该取哪个义？"
- "这句古文怎么翻译""帮我把这段文言文串讲一下"
- "通假字有哪些""这个词是古今异义吗""这里是不是词类活用"
- "江西中考文言文常考哪些词""帮我自测一下高频词"
- 备课、出题、答疑、学生自学任何涉及文言字词的场景。

## 核心能力（四种模式，按需切换）

### 模式一：单字 / 词查义 + 一词多义辨析

**流程**：
1. 在 `references/文言词表/` 中定位该字：
   - 先看 `江西省中考文言文高频词表.md`（104 个江西中考真题高频词，含义项 + 标注年份的真题例句）——命中即优先用此表，因为例句来自真实中考卷。
   - 未命中再查 `000中学文言文常用词常用义-合集.md`（775 词总索引，按出现频次五星排序）。
   - 更细的义项整理见 `文言文高频词义项整理_第3组.md`、`江西省中考真题文言文高频词义项整理_第2组.md`。
2. **若用户只问"什么意思"**：按中考/教材频度列出全部常见义项，每条配例。
3. **若用户给了具体句子（一词多义辨析）**：必须做"语境判定"——从这字的多个义项中**选定本句正确义项**，并写明判定依据（看句子成分、搭配、是否主谓之间等），不要只罗列义项。

**输出格式（字词卡片，锁死）**：

```
### 【字词卡片】<字>
- **读音 / 字形**：<音> （如有通假，注明"通 X"）
- **词性**：<实词/虚词/助词/代词…>
- **核心义项（按中考频度排序）**：
  1. <义项1> — 例：<例句>（<出处/年份>）
  2. <义项2> — 例：<例句>
  …
- **本句语境判定**：<若给了句子>本句取第 N 个义项，理由：<看 XX 结构/搭配>
- **中考频度**：<★★★★★ / 出现 N 次 / 江西 2014–2025>
- **易错提醒**：<与哪义易混、辨析窍门>
```

### 模式二：通假字 / 古今异义 / 词类活用 提示

本技能**不自建**这几类专项表，而是跨技能引用 `doc-review` 的权威笔记（单一数据源，避免重复维护）：

- 通假字 → `../../doc-review/references/古诗文基础/初中文言文通假字.md`
- 词类活用 → `../../doc-review/references/古诗文基础/初中文言文词类活用.md`、`文言文词类活用.md`
- 古今异义 / 一词多义 → `../../doc-review/references/古诗文基础/一词多义与古今异义.md`
- 文言虚词 → `../../doc-review/references/古诗文基础/初中常用文言虚词.md`、`初中文言文重点虚词.md`
- 文言实词 → `../../doc-review/references/古诗文基础/文言文实词.md`

**用法**：当用户问到某字是否通假 / 古今异义 / 活用，先查上述文件给出准确说法；并在字词卡片的"易错提醒"或串讲的"特殊用法"栏标注。

### 模式三：句子 / 课文逐字串讲

**流程**：
1. 逐字 / 逐词切分，对每个有解释价值的字调用模式一（优先用高频词表 / 合集）。
2. 标出特殊用法：通假、古今异义、词类活用、重点虚词（跨引模式二的文件）。
3. 给出整句通顺翻译（现代汉语句式，不硬译）。

**输出格式（串讲卡，锁死）**：

```
### 【串讲】<原句>
逐词标注：
- <字1>：<释义>（<如有特殊用法标注>）
- <字2>：<释义>
- …
特殊用法：<通假/活用/古今异义清单，无则写"无">
整句翻译：<通顺现代汉语>
```

### 模式四：江西中考 104 高频词 速查与自测

- **速查**：直接基于 `references/文言词表/江西省中考文言文高频词表.md`（104 词，2014–2025 真题例句），按词频或音序列出，标注每个词的中考出现次数与年份分布。
- **自测**：从高频词表中抽 N 个词，给出"请写出其在句中的含义 + 所属义项"的题，配答案（答案回查该表，保证与真题例句一致）。可限定年级 / 年份 / 实词虚词。

## 诚信与合规护栏

- **以权威为准**：释义基线来自统编教材与江西省中考真题。若遇争议或存疑释义，注明"以教材 / 江西省教育考试院当年发布为准"，不臆造。
- **真题例句须忠实**：引用中考例句时保留原句与年份标注，不篡改；本技能数据按 [数据年度更新SOP](../../docs/数据年度更新SOP.md) 每年核对。
- **不冒用学者署名**：本技能是工具性查词，不涉及王力/吕叔湘等学者视角模拟；如结合 `wangli-perspective` 等视角技能产出内容，对外发表须注明为"视角模拟、仅供参考"。
- **品牌声音**：对外呈现的讲解语气参考 [品牌声音与品牌规范](../../docs/品牌声音与品牌规范.md)（准确、干净、不浮夸）。

## 新增词目规范

如需补充词目，套用 `references/文言词表/词条标准模板.md`，保证 `字 / 音 / 词性 / 义项 / 例句(含出处年份) / 频度` 字段齐全，并与高频词表、合集的既有写法一致。

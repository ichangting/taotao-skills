# 中考真题深度分析与考情研判技能

## ⚠️ 重要说明

**本技能提供分析框架和模板，真题数据需由教师从权威来源获取后填入。**

**原因**：AI 无法保证真题数据的准确性和时效性，为避免误导，真题数据请以权威来源为准。

### 权威数据来源推荐

| 来源 | 网址/获取方式 | 可靠性 |
|------|--------------|--------|
| 江西省教育考试院 | www.jxeea.cn | ⭐⭐⭐⭐⭐ |
| 学科网 | www.zxxk.com（需会员） | ⭐⭐⭐⭐⭐ |
| 正确云 | www.zqy.com（需会员） | ⭐⭐⭐⭐ |
| 各地市教研室 | 内部资料 | ⭐⭐⭐⭐⭐ |
| 官方教辅 | 江西教育出版社《中考真题汇编》 | ⭐⭐⭐⭐⭐ |

## 技能定位

本技能为语文教师提供**中考真题分析框架、考情研判模板、备考方案生成工具**，帮助教师：
- 系统分析本地中考真题
- 精准把握命题规律
- 科学制定备考方案

**使用方式**：教师从权威来源获取真题数据后，填入本技能提供的模板，生成本地化考情分析报告。

## 使用流程

### Step 1: 获取真题数据

从以下权威来源获取江西省/本地市中考语文真题：

| 来源 | 获取方式 |
|------|----------|
| 江西省教育考试院 | www.jxeea.cn 官网下载 |
| 学科网 | 注册会员后下载 |
| 学校教研室 | 联系教研员获取 |
| 官方教辅 | 购买《江西省中考真题汇编》 |

### Step 2: 填入数据模板

将真题数据填入以下模板：
1. 真题数据录入表（每份试卷）
2. 考点分布统计表（多年对比）

### Step 3: 生成分析报告

基于填入的数据，生成：
1. 考点分布热力图
2. 命题趋势分析
3. 备考方案建议

### Step 4: 应用与实践

将分析结果应用于：
1. 日常教学调整
2. 复习计划制定
3. 模拟卷命制
4. 专题复习课设计

## 使用示例

### 示例 1：真题数据分析

**输入**（填入模板）：
```
年份：2024 年
板块一：语言知识与运用（24 分）
- 第 1 题：字音辨析，选择题，2 分，易
- 第 2 题：字形辨析，选择题，2 分，易
- 第 3 题：词语运用，选择题，2 分，中
...
```

**输出**：
- 考点分布统计表
- 高频考点 TOP10
- 命题趋势分析

### 示例 2：备考方案生成

**输入**：
```
学生层次：B 层（90-110 分）
复习阶段：第二轮
重点突破：现代文阅读、作文
可用课时：40 课时
```

**输出**：
- 专题复习计划表
- 训练重点建议
- 目标分值设定

## 注意事项

1. **数据真实性**：真题数据必须从权威来源获取，不得编造
2. **本地化**：以本省/本市考情为主，兼顾全国趋势
3. **时效性**：每年更新最新真题数据
4. **针对性**：根据学生层次提供差异化建议
5. **实用性**：方案可落地，建议可执行

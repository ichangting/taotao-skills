# 中考分析报告 · 阅读说明与诚信声明

本目录收录对 **2014–2025 年江西省中考语文真题** 的派生分析（趋势、五大题型专题、命题研究、评分标准、考情统计等），作为 `jiangxi-zhongkao-beikao` 技能的数据底座。

## 来源与性质
- 这些报告由多个 AI 助手（扣子 / 小浣熊 / IMA 等）基于历年真题**生成或辅助生成**，属"派生分析"，不是真题原卷本身。
- 本目录**不含任何完整真题原卷或官方解析原文**（原卷版权归江西省教育考试院，按仓库策略不随公开仓发布；教师可在本人私库取用）。
- 数据口径以 2021–2025 年为主；2026 年及以后的"命题预测"类报告**未收入**（属推测，不得当作既定事实）。

## 使用守则（务必遵守）
1. **以官方为准**：所有结论、趋势、考点归纳，均须以江西省教育考试院发布的当年真题与《评分标准》为准；本资料仅供参考。
2. **每年核对**：中考命题年年微调，使用前请按 `../../docs/数据年度更新SOP.md` 核对当年真题，更新考点与趋势判断。
3. **不臆造**：基于本报告设计备考方案时，不得把 AI 推断当作"官方定论"；凡涉及具体分值、题型、范文示例，须标注来源并可回溯到原卷。
4. **署名规范**：若对外发布基于本报告的内容，应说明"基于历年真题的派生分析"，不冒用官方或学者名义。

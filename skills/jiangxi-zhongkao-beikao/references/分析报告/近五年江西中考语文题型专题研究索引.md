# 近五年江西中考语文题型专题研究索引

> 研究对象：2021—2025 年江西省中考语文真题及解析。  
> 研究方式：按题型拆分为五个专题，分别分析题型结构、命题规律、典型设问与教学建议。

## 专题文档

1. 近五年江西中考语文语言文字运用专题研究
2. 近五年江西中考语文古代诗文阅读专题研究
3. 近五年江西中考语文现代文阅读专题研究
4. 近五年江西中考语文名著阅读专题研究
5. 近五年江西中考语文写作专题研究

## 总体判断

近五年江西省中考语文试卷形成了稳定的五板块结构：

| 板块 | 近五年分值趋势 | 命题关键词 |
|---|---:|---|
| 语言文字运用 | 10→12分 | 语境化、基础知识、真实表达 |
| 古代诗文阅读 | 稳定20分 | 课内积累、课外迁移、传统文化 |
| 现代文阅读 | 稳定30分 | 多文本、信息整合、文学审美、表达任务 |
| 名著阅读 | 10→8分 | 整本书阅读、人物情节、阅读心得 |
| 写作 | 稳定50分 | 成长体验、家国文化、以小见大 |

## 使用建议

- 备课时可按五大专题分别建立训练序列。
- 讲评真题时建议采用“题型—考点—方法—迁移练习”的路径。
- 作文与现代文阅读可联动训练：阅读中学习写法，作文中迁移表达。
- 文言文和名著阅读宜建立长期积累表，不宜考前突击。

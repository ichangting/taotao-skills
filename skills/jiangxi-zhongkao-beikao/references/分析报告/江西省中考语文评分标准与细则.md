# 江西省中考语文评分标准与细则

基于2021-2025年江西省中考语文真题解析及评分说明整理

---

## 一、试卷整体结构与分值分布

| 板块 | 分值 | 题量 | 建议用时 |
|------|:----:|:----:|---------|
| 语言文字运用 | 12分 | 6题 | 15分钟 |
| 古代诗文阅读 | 约20分 | 6-7题 | 25分钟 |
| 现代文阅读 | 约30分 | 8-9题 | 40分钟 |
| 名著阅读 | 8分 | 3题 | 10分钟 |
| 写作 | 50分 | 1题 | 60分钟 |
| **合计** | **120分** | **24-26题** | **150分钟** |

---

## 二、各板块评分细则

### （一）语言文字运用（12分）

#### 1. 字音字形填空题（2分）
- **评分标准**：每空1分，错字、别字、注音错误均不得分
- **注意事项**：书写必须规范，字迹潦草难以辨认按错误处理

#### 2. 词语运用选择题（2分）
- **评分标准**：选对得2分，选错得0分
- **命题特点**：侧重近义词辨析和成语在具体语境中的正确使用
- **常见错误类型**：望文生义、褒贬误用、对象错配、语境不符

#### 3. 病句辨析选择题（2分）
- **评分标准**：选对得2分，选错得0分
- **高频病句类型**：
  - 成分残缺（缺主语、缺宾语）
  - 语序不当
  - 搭配不当
  - 成分赘余
  - 句式杂糅

#### 4. 标点符号选择题（2分）
- **评分标准**：选对得2分，选错得0分
- **高频考点**：句号、顿号、逗号、分号的正确使用

#### 5. 语句衔接/排序题（2分）
- **评分标准**：选对得2分，选错得0分
- **解题要点**：关注关联词、代词指代、逻辑顺序（时间、空间、事理）

#### 6. 文体常识/口语交际题（2分）
- **评分标准**：选对得2分，选错得0分

---

### （二）古代诗文阅读（约20分）

#### 1. 名句默写（6分）
- **评分标准**：每空1分，错字、漏字、添字均不得分
- **题型**：3组，每组2句，以上下句默写为主，偶尔出现理解性默写
- **注意**：书写必须工整，连笔字如无法清晰辨认按错误处理

#### 2. 古诗词鉴赏（4-5分）
- **选择题（2分）**：选对得2分，选错得0分
  - 考查内容：诗句理解、意象分析、情感把握、手法判断

- **简答题（2-3分）**：
  - **评分标准**：
    - 3分题：观点明确（1分）+ 结合诗句具体分析（1分）+ 语言表达通顺（1分）
    - 2分题：观点明确（1分）+ 简要分析（1分）
  - **高频考点**：炼字赏析、情感分析、表现手法、画面描述

#### 3. 文言文阅读（10-11分）

| 题型 | 分值 | 评分标准 |
|------|:----:|---------|
| 文言实词解释 | 3分 | 每空1分，意思对即可 |
| 句子翻译 | 4分 | 每句2分，关键词译对1分，句子通顺1分 |
| 内容理解/断句 | 3-4分 | 断句正确每处0.5-1分；内容理解题观点+分析各占分值 |

- **关键词翻译评分原则**：
  - 必须准确翻译出通假字、古今异义、词类活用等特殊用法
  - 特殊句式（判断句、倒装句、省略句）必须体现出来
  - 直译为主，意译为辅

---

### （三）现代文阅读（约30分）

#### 1. 议论文/说明文阅读（6-9分）
- **选择题（2-3分）**：选对得分，选错0分
  - 考查：论点判断、论据与论点关系、内容理解

- **简答题（4-6分）**：
  - **3分题**：观点明确（1分）+ 结合文本分析（1分）+ 表达清晰（1分）
  - **4分题**：两个要点，每个要点2分
  - **考查**：论证方法及作用、信息提取概括、结合生活谈思考

#### 2. 非连续性文本阅读（6-7分）
- **选择题（2-3分）**：选对得分
- **简答题（4分）**：
  - 信息提取与概括：每点1-2分，答对2-4点满分
  - 提建议/写说明性文字：建议合理2分，表述清晰2分
  - 2025年新增"写说明性文字"（80字左右）：内容准确4分，字数不够或表达不清酌情扣分

#### 3. 文学类文本阅读（14-17分）
这是阅读题中分值最高、评分最细致的板块

| 考点 | 分值 | 评分标准 |
|------|:----:|---------|
| **人物形象分析** | 3-4分 | 答出性格特点2分 + 结合文本分析2分，答对2-3点满分 |
| **内容概括** | 3-4分 | 每概括一个要点1-2分，要点完整得满分 |
| **语句赏析** | 4-5分 | 指出手法1分 + 分析表达效果2分 + 结合主题1分 |
| **环境描写作用** | 3分 | 答出1-2个作用得2分，结合文本分析再得1分 |
| **主题理解** | 3-4分 | 表层含义2分 + 深层含义2分 |
| **写作借鉴/启发** | 3分 | 结合文本内容1分 + 联系写作实际2分 |

**通用评分原则**：
- 采点给分，言之有理即可酌情给分
- 结合文本具体内容是得分关键，仅有观点无分析最多得一半分
- 观点明确，条理清晰，语言通顺

---

### （四）名著阅读（8分）

| 题型 | 分值 | 评分标准 |
|------|:----:|---------|
| 选择题 | 2分 | 选对得2分 |
| 简答题一 | 3分 | 内容准确2分 + 表达清晰1分 |
| 简答题二 | 3分 | 内容准确2分 + 表达清晰1分 |

**简答题评分细则**（80-100字）：
- **一等（3分）**：观点明确，结合名著具体内容，分析到位，语言流畅，字数达标
- **二等（2分）**：观点较明确，有一定分析，内容基本正确，语言较通顺
- **三等（1分）**：仅有观点，无具体分析，或内容不准确
- **四等（0分）**：未作答，或内容与题目无关

---

### （五）写作（50分）

#### 作文评分标准（五等评分制）

| 等次 | 分值 | 评分标准 |
|------|:----:|---------|
| **一等（优秀）** | 45-50分 | 立意深刻，中心突出，内容充实，感情真挚，结构严谨，语言流畅，符合文体要求 |
| **二等（良好）** | 38-44分 | 中心明确，内容较充实，感情真实，结构完整，语言通顺，大体符合文体要求 |
| **三等（中等）** | 30-37分 | 中心基本明确，内容单薄，感情基本真实，结构基本完整，语言基本通顺，基本符合文体要求 |
| **四等（较差）** | 21-29分 | 中心不明确，内容空洞，感情虚假，结构混乱，语病较多，不符合文体要求 |
| **五等（差）** | 0-20分 | 严重偏离题意，或文理不通，或结构残缺，或字数严重不足 |

#### 具体评分细则

**1. 基础等级（40分）**

| 项目 | 评分要点 |
|------|---------|
| **审题立意（15分）** | - 准确理解题意，立意明确深刻（13-15分）<br>- 基本理解题意，立意较明确（10-12分）<br>- 偏离题意，立意不明确（6-9分）<br>- 严重偏离题意（0-5分） |
| **内容选材（10分）** | - 选材真实典型，内容充实，有真情实感（9-10分）<br>- 选材较恰当，内容较充实，感情真实（7-8分）<br>- 内容单薄，选材一般，感情基本真实（5-6分）<br>- 内容空洞，选材不当，感情虚假（0-4分） |
| **结构布局（8分）** | - 结构严谨，层次分明，过渡自然（7-8分）<br>- 结构完整，条理清楚（5-6分）<br>- 结构基本完整，条理基本清楚（3-4分）<br>- 结构混乱，残缺不全（0-2分） |
| **语言表达（7分）** | - 语言流畅，有文采，表达准确（6-7分）<br>- 语言通顺，表达较准确（4-5分）<br>- 语言基本通顺，有少量语病（2-3分）<br>- 语病较多，表达不清（0-1分） |

**2. 发展等级（10分）**

- **深刻**：透过现象深入本质，揭示事物内在的因果关系，观点具有启发性（3分）
- **丰富**：材料丰富，论据充实，形象丰满，意境深远（2分）
- **有文采**：用词贴切，句式灵活，善于运用修辞手法，文句有表现力（3分）
- **有创意**：见解新颖，材料新鲜，构思精巧，推理想象有独到之处，有个性特征（2分）

#### 扣分项

- **字数不足**：每少50字扣1分，不足600字降至五等
- **错别字**：每1个错别字扣1分，重复不计，最多扣3分
- **标点错误**：标点错误较多或模糊不清的，酌情扣1-2分
- **无题目**：扣2分
- **套作、抄袭**：套作酌情扣分，抄袭的判零分

#### 江西省中考作文评分特点

1. **注重真情实感**：2024年明确要求"有益身心健康的活动"素材，强调真实经历
2. **关注小我与大我的结合**：2025年作文题《我也是一张"中国名片"》，要求将个人与国家关联
3. **材料引导越来越具体**：从单纯命题到提供名言、事迹作为引导
4. **反套作意识强**：过于模板化、套路化的作文很难进入一等卷
5. **书写占比较高**：工整清晰的书写在实际阅卷中具有隐形优势

---

## 三、整体评分原则

1. **采点给分与整体把握相结合**：主观题既看要点数量，也看整体质量
2. **鼓励创造性思维**：言之有理、有独立见解的答案可酌情加分
3. **容错机制**：要点答错不倒扣分，答对就给分
4. **书写要求**：字迹潦草、难以辨认的，主观题酌情扣1-2分
5. **开放性试题**：不设统一标准答案，符合题意、言之有理即可得分

---

## 四、备考建议

1. **基础题**：追求满分，字音字形、默写等主观题要特别注意书写规范
2. **阅读题**：强化"观点+文本分析"的答题模式，注意分点作答
3. **名著题**：不要死记硬背，要真正理解人物和主题，练习80字左右的精炼表达
4. **作文**：
   - 准备3-5个真实的个人经历素材
   - 练习"具体事例+升华主题"的结构模式
   - 开头结尾要精心打磨，快速入题，卒章显志
   - 注意书写工整，避免涂改

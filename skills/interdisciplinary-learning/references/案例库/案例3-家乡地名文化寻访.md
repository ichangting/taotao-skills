# 案例 3：七年级"家乡地名文化寻访"（8 课时）⭐ 初中段

## 一、基本信息

| 项目 | 内容 |
|------|------|
| 主题 | 家乡地名文化寻访 |
| 学段 | 第四学段（七年级） |
| 课时 | 8 课时 + 课外 2 周寻访 |
| 跨学科 | 语文 + 历史 + 地理 + 道德与法治 |
| 课标对应 | 第四学段跨学科学习：参与文化体育活动，体验、感知、传承中华优秀传统文化 |

## 二、学习目标

**语文核心素养**：
- 语言运用：能撰写访谈提纲、考察笔记与汇报文稿
- 文化自信：在寻访中理解地名承载的乡土文化与家国情怀

**跨学科能力**：
- 历史：考证地名沿革与历史人物掌故
- 地理：在地图上标注地名空间分布
- 道德与法治：体会家乡建设与公民责任

## 三、活动流程

| 阶段 | 课时 | 活动 | 成果 |
|------|------|------|------|
| 阶段一 | 1 课时 | 确定寻访地名，分组选题 | 寻访计划表 |
| 阶段二 | 课外 2 周 | 实地寻访、访谈长辈、记录 | 考察笔记、访谈录音 |
| 阶段三 | 2 课时 | 整理资料，撰写推文/汇报稿 | 公众号推文、汇报文稿 |
| 阶段四 | 2 课时 | 地图标注与成果展 | 地名文化地图、展示汇报 |
| 阶段五 | 1 课时 | 评价反思 | 反思单 |

## 四、访谈提纲模板

```
【家乡地名文化访谈提纲】

访谈对象：_______    与地名关系：_______
访谈时间：_______    地点：_______

1. 这个地名您小时候叫什么？有没有别的叫法？
2. 它大概是什么时候、因为什么得名的？
3. 您记得和这个地方有关的老故事或老物件吗？
4. 这些年这个地方最大的变化是什么？
5. 您最想把这个地名里的什么传给下一代？
```

## 五、评价标准

| 维度 | 优秀 | 合格 |
|------|------|------|
| 考察笔记 (40%) | 记录详实，有亲身观察与追问 | 有基本记录 |
| 访谈文稿 (30%) | 问题设计合理，表达清晰 | 能完成口头/书面汇报 |
| 文化地图 (30%) | 标注准确，有文化解读 | 标注基本完整 |

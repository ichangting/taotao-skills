# 案例 2：五年级传统节日文化探究（8 课时）⭐ 小学段

## 一、基本信息

| 项目 | 内容 |
|------|------|
| 主题 | 传统节日文化探究 |
| 学段 | 第三学段（五年级） |
| 课时 | 8 课时 + 课外 1 周 |
| 跨学科 | 语文 + 道德与法治 + 艺术 + 劳动 |
| 课标对应 | 第三学段跨学科学习（1）：参与传统文化活动，体验、感知、传承中华优秀传统文化 |

## 二、学习目标

**语文核心素养**：
- 语言运用：能撰写节日文化研究报告，进行口头汇报
- 文化自信：了解传统节日文化内涵，增强文化认同

**跨学科能力**：
- 道德与法治：理解节日的文化意义
- 艺术：制作节日手抄报、手工作品
- 劳动：体验传统节日食品制作

## 三、研究方向

| 节日 | 研究内容 | 成果形式 |
|------|----------|----------|
| 春节 | 起源、习俗、美食、春联 | 手抄报、春联创作 |
| 清明节 | 起源、习俗、诗词 | 诗词朗诵、祭奠活动 |
| 端午节 | 屈原故事、粽子制作、赛龙舟 | 手工粽子、故事讲述 |
| 中秋节 | 起源、习俗、诗词、月饼 | 月饼制作、诗词会 |
| 重阳节 | 起源、习俗、敬老活动 | 敬老活动、手抄报 |

## 四、活动流程

| 阶段 | 课时 | 活动 | 成果 |
|------|------|------|------|
| 阶段一 | 1 课时 | 主题导入，分组选题 | 研究计划表 |
| 阶段二 | 2 课时 | 资料查阅，整理汇编 | 资料汇编 |
| 阶段三 | 2 课时 + 课外 | 体验活动（食品制作等） | 体验记录、照片 |
| 阶段四 | 2 课时 | 报告撰写，手抄报制作 | 研究报告、手抄报 |
| 阶段五 | 1 课时 | 成果展示，评价反思 | 展示汇报 |

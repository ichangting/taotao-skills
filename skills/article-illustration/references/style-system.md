## 概览
本文档定义了18种标准配图风格,每种风格都有明确的视觉特征、情感基调和适用场景。智能体在规划配图时必须从此体系中选择合适的风格。其中前12种为通用风格,后6种为故事专属插画风格。

---

## notion简约

**描述**:极简手绘线条、干净清爽,类似Notion文档的视觉风格

**视觉特征**:
- 简洁的黑白线条
- 干净的白色或浅灰背景
- 极简的图标和符号
- 留白充足,视觉清爽
- 无多余装饰元素

**情感基调**:专业、简洁、现代、清爽

**最佳适用场景**:
- 知识分享(SaaS工具、生产力工具)
- 技术文档(技术博客、API文档)
- 教程类内容(操作指南、教程)
- 职场内容(工作方法、效率提升)

**配色建议**:
- 主色:黑色、深灰色
- 背景:白色、浅灰色(#F5F5F5)
- 点缀色:单一强调色(可选)

**避免元素**:
- 复杂的渐变和阴影
- 过多的装饰元素
- 鲜艳的配色
- 复杂的背景纹理

**关键词示例**:
- 极简线条、简约风格、干净清爽、黑白线条、notion风格、手绘线条、简洁

## 温暖手绘

**描述**:柔和笔触、暖色调、亲和力强,营造温暖亲切的视觉感受

**视觉特征**:
- 柔和的手绘笔触
- 温暖的色调(暖黄、橙色、粉色)
- 亲切友好的氛围
- 圆润的线条和形状
- 自然的质感

**情感基调**:温暖、亲切、友好、柔和

**最佳适用场景**:
- 个人成长(成长故事、心得分享)
- 生活方式(生活分享、日常记录)
- 教育(教育内容、学习分享)
- 情感类内容(情感故事、心灵鸡汤)

**配色建议**:
- 主色:暖黄色(#FFE4B5)、柔和粉色(#FFB6C1)
- 背景:米白色(#FFFAF0)、淡黄色
- 点缀色:橙色、棕色系

**避免元素**:
- 冷色调
- 生硬的直线和几何形状
- 过于复杂的细节
- 刺眼的对比色

**关键词示例**:
- 温暖手绘、柔和笔触、暖色调、亲切风格、手绘风格、温暖感、亲和力

- 阴影和暗部突出
- 神秘的氛围营造
- 不确定性和紧张感
- 可以包含谜题元素

**情感基调**:悬疑、神秘、紧张、推理

**最佳适用场景**:
- 悬疑小说(悬疑、推理故事)
- 侦探故事(侦探、破案)
- 惊悚故事(惊悚、恐怖)
- 神秘故事(神秘、未解之谜)
- 推理游戏(推理、解谜)

**配色建议**:
- 主色:深蓝色(#1A1A2E)、深紫色(#2D132C)
- 背景:深色背景
- 点缀色:血红色(#E94560)、荧光色(线索)

**避免元素**:
- 过于明亮的色调
- 轻松愉快的元素
- 缺乏神秘感的平凡场景
- 过于鲜艳可爱的色彩

**关键词示例**:
- 悬疑风格、推理风格、神秘插画、悬疑氛围、侦探风格、惊悚风格、神秘感

## 风格选择指南

如何根据文章特点选择合适的风格:

**通用风格**:
1. **知识分享/技术文档** → notion简约
2. **个人成长/情感内容** → 温暖手绘
3. **技术架构/系统设计** → 蓝图技术
4. **小红书/年轻群体** → 马卡龙清新
5. **商业报告/PPT** → 商务专业
6. **传统文化/历史文学** → 国风水墨
7. **AI/科技趋势/游戏** → 暗黑科技
8. **散文/旅行/艺术** → 水彩文艺
9. **通用内容/互联网产品** → 扁平插画
10. **科普/故事/教育** → 知识漫画风
11. **教程/干货/读书笔记** → 视觉笔记风
12. **技术草稿/头脑风暴** → 白板风

**故事专属插画风格**:
13. **现实题材/悬疑推理** → 写实插画风格
14. **童话/奇幻/儿童故事** → 梦幻童话风格
15. **怀旧故事/历史题材** → 复古怀旧风格
16. **悬疑推理/惊悚** → 悬疑推理风格
17. **治愈系/情感故事** → 治愈系风格
18. **史诗奇幻/冒险英雄** → 史诗幻想风格

> 适用范围：中学语文文言文常用字、常用词、虚词、实词、通假字、词类活用等单字词条。

## 字段说明

### 1. 笔记属性

建议保留这些字段：

```yaml
字: 
拼音: 
繁体: 
词类: 
星级: 
中考频率: 
义项数量: 
是否虚词: 
是否通假字: 
是否词类活用: 
标签:
  - 文言文
  - 常用字
```

不建议把“教材来源”放在笔记属性里。原因是：一个字往往有多个义项，每个义项又可能有多个例句，例句来源可能分布在不同教材册次、不同课文和不同试题中。把教材来源放到整篇笔记属性中，会造成信息失真。

更合理的做法是：

- 整篇笔记属性只记录这个字本身的稳定信息。
- 教材册次、课文来源、真题来源放到每条例句下。

### 2. 常用义项

“常用义项”是整篇词条的核心部分。每个义项建议包含：

- 释义
- 词性 / 用法
- 例句
- 出处
- 教材位置
- 译文
- 考点提示
- 识别方法

这样可以同时满足教师备课、学生复习和题库生成。

### 3. 义项脉络

“义项脉络”放在“常用义项”之后，用来整理这个字各义项之间的关系。

它重点回答四个问题：

1. 这个字的本义是什么？
2. 哪些意思是从本义引申出来的？
3. 哪些意思属于假借义或通假义？
4. 哪些意思属于比喻义？

这个栏目不重复罗列义项，而是解释义项之间的生成关系。它的教学价值在于帮助学生把“一词多义”从机械记忆变成有逻辑的理解。

建议写法：

```markdown
## 三、义项脉络

这个字的本义是……，后来由……引申出……。在中学文言文中，最常考的是……。
```

如果某个字义项很少，也可以简写为一段话，不必强行填表。

### 4. 为什么删除“一词多义辨析”栏目

原来的“一词多义辨析”容易和“常用义项”重复。现在把辨析功能放进两个位置：

- 在“常用义项”中，通过“考点提示”和“识别方法”处理具体义项的辨析。
- 在“义项脉络”中，说明各义项之间的来源和关系。

这样既避免重复，又能保留一词多义教学中真正有价值的部分。

如果确实需要横向比较，可以在具体词条中临时增加“义项对照表”，但不作为标准模板的固定栏目。

### 5. 教材位置字段

建议在每条例句下使用：

```markdown
**【教材位置】**  
七年级上 / 八年级下 / 九年级上 / 课外 / 中考真题 / 模拟题
```

如果能进一步明确，可以写成：

```markdown
**【教材位置】**  
八年级下，《桃花源记》
```

或者：

```markdown
**【教材位置】**  
2023年江西中考文言文阅读
```

### 6. 星级建议

星级可以表示教学优先级，而不是单纯表示字的难度：

| 星级 | 含义 |
|---|---|
| 五星 | 中考高频，必须掌握 |
| 四星 | 课内高频，常用于翻译和解释 |
| 三星 | 常见但考查频率中等 |
| 二星 | 较少单独考查，作为补充 |
| 一星 | 低频，保留备查 |

### 7. 推荐使用方式

新建或修订词条时，建议按以下顺序处理：

1. 先写“核心结论”，明确这个字最值得学生掌握的意思。
2. 再整理“常用义项”，每个义项至少配一个可靠例句。
3. 然后写“义项脉络”，说明本义、引申义、假借义、比喻义之间的关系。
4. 再补充“特殊用法”，包括通假、活用和固定搭配。
5. 最后写“中考常见考法”和“易错提醒”。
6. 如果信息暂时不完整，宁可标注“待核”，不要直接生成不确定内容。

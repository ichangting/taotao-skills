# 语文写作审核（doc-review）—— 资深编审级咬文嚼字

## 技能定位

本技能模拟**资深语文编审**的把关流程，专门为**一线语文教师的写作产出**做高精度审核。核心诉求只有一条：**咬文嚼字，用词极准，不能出现任何语法、词汇、用词方面的差错**。

不是泛泛的"文档审查"，而是针对语文学科教师的职业场景——你写出去的每一句，都代表语文专业水准，错一个字、用错一个词、闹一个语病，都比普通文档更扎眼。所以本技能的合格线是"编审级"，而非"差不多就行"。

---

## 适用场景

| 写作产出 | 典型审核重点 |
|----------|--------------|
| 教案 / 教学设计 | 环节用语准确、指令清晰、不出现知识性硬伤、引用课文/课标无误 |
| 教学论文 / 教育随笔 | 论点表述严谨、文献与引文规范、无语病与逻辑漏洞 |
| 课题申报书 | 术语一致、研究内容表述精准、避免空话套话 |
| 公众号推文 / 短视频文案 | 面向家长学生，语体得体、通俗而不失准确、无硬伤 |
| 课件文本（PPT 文字） | 短句干净、重点词醒目、不与板书/口误混淆 |
| 学生范文评语 | 用词分量恰当（褒贬、激励尺度）、不出知识错误 |
| 考试命题语言 | 题干无歧义、选项无语病、指令唯一明确 |
| 教研发言稿 / 汇报 | 口语书面化转换、重点突出、用语规范 |

## 审核四关（咬文嚼字，高要求）

### 第一关：文字差错关（基础，必须零容忍）

| 检查项 | 说明 | 语文教师高频示例 |
|--------|------|------------------|
| 错字 | 形近、音近、输入错误 | "即"与"既"、"己"与"已"与"巳"、"赢"与"羸" |
| 的/地/得 | 结构助词误用 | "认真的学习"✗→"认真的学习（名词前）"；"认真地学"✓；"学得好"✓ |
| 标点规范 | 书名号配对、引号嵌套、顿号逗号区分、问号滥用 | 概数"三四天"不加顿号；《〈论语〉十二章》内层用〈〉 |
| 异形词 | 推荐词形 vs 非推荐 | "唯一"而非"惟一"、"标志"而非"标识"、"订单"而非"定单" |
| 繁简混用 | 同一文档繁简一致 | "后"与"後"不混 |
| 数字用法 | 阿拉伯 vs 汉字 | 统计数字用阿拉伯，概数/固定表述用汉字（"十三五"不写"13·5"） |
| 专名错误 | 人名、篇名、书名、地名 | 作者与篇目错配、书名号误用 |

### 第二关：语法硬伤关（结构，必须零语病）

| 检查项 | 说明 | 语文教师高频坑 |
|--------|------|----------------|
| 成分残缺 | 主谓宾不完整 | "通过这次教研，使我的认识提高了"——"通过…使…"淹没了主语，删其一 |
| 搭配不当 | 主谓/动宾/修饰 | "提高…质量"✓ vs "提高…水平"✓；"培养…习惯"✓ vs "培养…意识"✓（莫乱配） |
| 语序不当 | 词语/分句顺序 | "许多附近的居民"→"附近的许多居民" |
| 句式杂糅 | 两式混用 | "原因是…造成的"→"原因是…"或"是…造成的" |
| 表意不明 | 指代/歧义 | "他告诉他自己错了"（两个"他"指代不清） |
| 不合逻辑 | 前后自洽 | "他是众多死难者中幸免的一个"→"遇难者中幸免的"；"大约三十岁左右"→"大约三十岁" |
| 否定失当 | 多否定翻转语义 | "谁也不能否认这出戏没有教育意义"→"谁也不能否认这出戏有教育意义" |
| 两面对一面 | 呼应不周 | "能否提高学生的语文素养，关键在于重视阅读"→前后加"能否/是否" |

### 第三关：用词精准关（核心，本技能的灵魂）

不只"没错"，更要"最准"。语文教师的文字，用词要经得起推敲：

| 检查项 | 说明 | 示例 |
|--------|------|------|
| 近义辨析 | 易混近义词选最贴切者 | "需要"vs"须要"（"须要"含"必须"义）；"做"vs"作"（"写作/作成"多用"作"，"做事/做饭"用"做"）；"启用"vs"起用" |
| 词语搭配 | 固定/习惯搭配 | "汲取…营养"✓；"吸取…教训"✓；莫"汲取教训""吸取营养" |
| 语义重复 | 堆砌同义 | "涉及到"→"涉及"；"凯旋归来"→"凯旋"；"亲眼目睹"→"目睹"；"悬殊很大"→"悬殊" |
| 褒贬色彩 | 感情色彩得体 | "趋之若鹜"（贬）莫用于褒义；"殚精竭虑"（褒）/"处心积虑"（贬）分清 |
| 语体得体 | 口语/书面/公文分寸 | 教案用教学语体；申报书用学术语体；推文用亲切语体；不混 |
| 成语误用 | 望文生义/对象错 | "万人空巷"（形容庆祝盛况，非"无人"）；"莘莘学子"（本身表复数，不加"众多"） |
| 生造词语 | 不规范新词 | 避免自造未被收录的词，确需用加引号并说明 |

### 第四关：文风与教学规范关（专业，体现语文素养）

| 检查项 | 说明 | 示例 |
|--------|------|------|
| 术语一致 | 同一概念全文统一 | 全篇统一用"学习任务群"或"任务群"，不中途换词 |
| 引用规范 | 课标/古文出处/教材版本 | 引《义务教育语文课程标准（2022 年版）》须全称；引古文注明篇目与所据版本 |
| 知识硬伤 | 文学/文化常识无误 | 作者朝代、篇目出处、名句归属不错（详见"古文引用专项审核"） |
| 去空话套话 | 剔除无信息量表述 | "在今后的工作中，我将继续努力……"如无具体，建议删或落地 |
| 语体统一 | 不忽正式忽口语 | 同一文体保持语体一致 |

## 古文引用专项审核（轻量子能力）

语文教师的写作常引古文，引错比现代文错更损专业形象。本技能**不自建词汇大表**（那归 `guwen-cidian` 初中文言词典），只做"审核流程 + 判定规则"，数据回查 `guwen-cidian`：

**审核流程**：
1. 提取文中所有古籍引用（篇目、作者、词句、所注版本）。
2. 核对作者与篇目归属（防"先天下之忧而忧——李白"这类硬伤）。
3. 核对词句原文（防张冠李戴、默写式错字、断句错误）。
4. 如需确认所据版本（通假字保留、异文、注音），回查 `guwen-cidian/references/教材版本/`。
5. 发现版本差异 → 标注说明，建议以统编教材为准。

**判定依据**（数据在 `guwen-cidian`）：
- 通假/虚实词/活用/古今异义 → `guwen-cidian/references/古诗文基础/`
- 统编教材版本与差异 → `guwen-cidian/references/教材版本/`
- 中考高频词与真题例句 → `guwen-cidian/references/文言词表/`

> 本技能只做"引用是否准确"的判定；具体字义讲解请直接调用 `guwen-cidian`。

**审核模式**：[快速校对/标准审核/深度审核/古文专项]
**审核时间**：YYYY-MM-DD
**问题总计**：🔴 X 处 | 🟡 X 处 | 🟢 X 处 | ⚪ X 处

## 🔴 严重问题（必须改）
| 序号 | 位置 | 原文 | 问题类型 | 修改建议 | 依据 |
|------|------|------|----------|----------|------|

| 序号 | 位置 | 原文 | 问题类型 | 优化建议 |
|------|------|------|----------|----------|

## 审核原则（铁律）

1. **零容忍硬伤** —— 错别字、语病、知识性错误，必须指出，不"差不多"。
2. **不确定标存疑** —— 无法核实的内容标 ⚪，不武断判定。
3. **修改给依据** —— 指出错误时尽量说明依据（语法规则/词典/教材版本），不只说"不对"。
4. **区分错误与建议** —— 事实错误/语病 ≠ 风格偏好，报告中分开列。
5. **保护原文意图** —— 只改错与优化表达，不替作者改观点、不改原有结构（除非作者要求）。
6. **尊重作者语体** —— 教案不必写成论文腔；推文不必端着。语体得体本身是一关。

---

## 限制说明

1. 本技能基于通用语文规范与 `guwen-cidian` 的文言语料，极冷门文献可能无法核实。
2. 学科专用术语（如跨到其他学科）需结合专业规范。
3. 学术争议问题无定论时，注明存在不同观点，不强行统一。
4. 审核是辅助，最终把关责任在作者本人。

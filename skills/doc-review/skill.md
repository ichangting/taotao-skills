# 文档审查技能 (Doc Review) - 资深编审版

## 技能定位

本技能模拟**资深编审**的审查流程，提供多层次、系统化的文档审查服务。不仅检查表面错误，更关注内容质量、逻辑一致性和专业规范性。

---

## 五层审查体系

### 第一层：文字审查（基础层）

| 检查项 | 说明 | 示例 |
|--------|------|------|
| 错别字 | 同音字、形近字、输入错误 | "的/地/得"混用、"已/己"混淆 |
| 标点符号 | 标点使用规范性 | 书名号配对、引号嵌套、顿号逗号区分 |
| 异形词 | 推荐词形 vs 非推荐词形 | "唯一"而非"惟一"、"标志"而非"标识" |
| 繁简混用 | 繁简体不一致 | 同一文档中"后"与"後"混用 |
| 数字用法 | 阿拉伯数字 vs 汉字数字规范 | 统计数字用阿拉伯数字，概数用汉字 |

### 第二层：语法审查（结构层）

| 检查项 | 说明 | 示例 |
|--------|------|------|
| 成分残缺 | 主谓宾是否完整 | "通过这次学习，使我受益匪浅"（缺主语） |
| 搭配不当 | 主谓、动宾、修饰搭配 | "提高...水平"✓ vs "提高...质量"✗ |
| 语序不当 | 词语/分句顺序是否合理 | "许多附近的居民"→"附近的许多居民" |
| 句式杂糅 | 两种句式混用 | "原因是...造成的"→"原因是..."或"是...造成的" |
| 指代不明 | 代词指代是否清晰 | "他告诉他自己错了"（两个"他"指代不清） |
| 逻辑矛盾 | 前后表述是否自洽 | "几乎全部"vs"全部" |

### 第三层：格式审查（规范层）

| 检查项 | 说明 | 示例 |
|--------|------|------|
| 标题层级 | 标题编号是否连续、层级是否清晰 | 一、→(一)→1.→(1) 层级递进 |
| 段落格式 | 段落缩进、空行是否一致 | 首行缩进 2 字符，段间空行统一 |
| 引用格式 | 引用标注是否规范 | 脚注/尾注格式、参考文献格式 |
| 列表格式 | 列表项格式是否统一 | 项目符号/编号风格一致 |
| 图表编号 | 图表编号是否连续、引用是否对应 | "见图 1"是否对应实际图 1 |
| 日期格式 | 日期书写是否统一 | "2024 年 1 月 1 日"vs"2024-01-01" |

### 第四层：内容审查（事实层）

| 检查项 | 说明 | 示例 |
|--------|------|------|
| 文言文核对 | 出处、作者、词句准确性 | 见下方「文言文知识库」 |
| 历史事实 | 人物、事件、时间、地点 | 历史人物生卒年、事件发生时间 |
| 科学常识 | 科学术语、公式、数据 | 物理常数、化学方程式 |
| 地理信息 | 地名、行政区划、位置关系 | 省市隶属关系、地理位置描述 |
| 人物信息 | 姓名、职务、机构名称 | 现任/历任职务、机构全称 |
| 数据一致性 | 文中数据是否前后一致 | 总数与分项之和是否相等 |
| 时间线逻辑 | 事件顺序是否合理 | 因果关系时间顺序 |

### 第五层：风格审查（专业层）

| 检查项 | 说明 | 示例 |
|--------|------|------|
| 语体一致 | 正式/非正式语体是否统一 | 公文用正式语体，避免口语化 |
| 术语统一 | 同一概念用词是否一致 | 全文统一用"用户"或"客户" |
| 人称一致 | 叙述人称是否统一 | 第一人称/第三人称不混用 |
| 语气适当 | 语气是否符合文档类型 | 公文庄重、通知明确、说明客观 |
| 冗余精简 | 是否有重复、啰嗦表述 | "涉及到"→"涉及"、"凯旋归来"→"凯旋" |
| 敏感表述 | 是否有不当或敏感表述 | 政治、民族、宗教等敏感话题 |

## 审查模式

用户可指定审查深度：

| 模式 | 覆盖层级 | 适用场景 |
|------|----------|----------|
| 快速校对 | 第一层（文字） | 日常文档、即时通讯内容 |
| 标准审查 | 第一 + 二层（文字 + 语法） | 工作报告、邮件、一般文章 |
| 深度审查 | 第一 + 二 + 三层（文字 + 语法 + 格式） | 正式文件、出版物、学术论文 |
| 全面审查 | 全部五层 | 重要公文、教材、对外发布内容 |
| 专项审查 | 指定某一层或某几项 | 针对性检查（如仅核对文言文引用） |

## 审查流程

### Step 1: 接收文档

- 接收用户提供的文档内容（文本、文件、截图 OCR）
- 询问或确认文档类型（公文/论文/教材/宣传材料等）
- 询问或确认审查模式

### Step 2: 预扫描

- 快速浏览全文，了解文档结构和主题
- 识别文档类型，确定适用的规范标准
- 标记明显可疑之处

### Step 3: 分层审查

按五层体系逐层审查，每层独立记录问题：

```
第一层审查：文字问题 X 处
第二层审查：语法问题 X 处
第三层审查：格式问题 X 处
第四层审查：事实问题 X 处
第五层审查：风格问题 X 处
```

### Step 4: 问题分级

将发现的问题按严重程度分级：

| 级别 | 标识 | 说明 | 处理建议 |
|------|------|------|----------|
| 严重 | 🔴 | 事实错误、严重语病、敏感问题 | 必须修改 |
| 中等 | 🟡 | 语法问题、格式不规范 | 建议修改 |
| 轻微 | 🟢 | 用词不够精炼、风格小瑕疵 | 可选优化 |
| 存疑 | ⚪ | 无法确定是否正确 | 需人工核实 |

### Step 5: 输出审查报告

输出结构化审查报告（见下方模板）

## 常见语法错误速查

### 病句类型

| 类型 | 错误示例 | 修正 |
|------|----------|------|
| 成分残缺 | 通过这次学习，使我受益匪浅 | 这次学习使我受益匪浅 |
| 搭配不当 | 他的事迹值得我们可歌可泣 | 他的事迹可歌可泣 |
| 语序不当 | 许多附近的居民前来观看 | 附近的许多居民前来观看 |
| 句式杂糅 | 原因是因为天气不好 | 原因是天气不好 / 是因为天气不好 |
| 表意不明 | 他看见她笑了 | 他看见她，她笑了 / 他看见她，他笑了 |
| 不合逻辑 | 他是众多死难者中幸免的一个 | 他是众多遇难者中幸免的一个 |
| 重复赘余 | 他大约三十岁左右 | 他大约三十岁 / 他三十岁左右 |
| 否定不当 | 谁也不能否认这出戏没有教育意义 | 谁也不能否认这出戏有教育意义 |

## 标点符号规范速查

### 常见错误

| 错误 | 正确 | 说明 |
|------|------|------|
| 书名号内再用书名号 | 《〈论语〉十二章》 | 外层用《》，内层用〈〉 |
| 引号内句末标点 | 他说："你好。" | 完整引用，标点在引号内 |
| 引号内句末标点 | 他说的"你好"很温暖 | 部分引用，标点在引号外 |
| 顿号用于概数 | 三、四天 | 三四天（概数不用顿号） |
| 顿号用于并列谓语 | 他唱歌、跳舞、绘画 | 他唱歌，跳舞，绘画（谓语用逗号） |
| 问号滥用 | 我不知道他是谁？ | 我不知道他是谁。（陈述句） |
| 省略号与"等"连用 | 苹果、香蕉……等 | 苹果、香蕉等 / 苹果、香蕉…… |

## 使用示例

### 示例 1：全面审查

**用户**：请帮我全面审查这份公文草稿

**审查报告**：
```markdown
# 文档审查报告

**文档类型**：公文
**审查模式**：全面审查
**问题总计**：🔴 2 处 | 🟡 3 处 | 🟢 1 处

## 🔴 严重问题

| 1 | 第 2 段 | "根据《论语》记载，孔子曰..." | 出处错误 | 《论语》是孔子弟子及再传弟子编纂，非孔子自著 | 应改为"《论语》记载孔子曰" |

## 🟡 中等问题

| 1 | 第 3 段 | "涉及到多个部门" | 冗余 | "涉及多个部门" |
```

### 示例 2：文言文专项核对

**用户**：请核对这份试卷中的文言文引用

**审查**：逐条核对出处、作者、词句，输出核对结果表

## 注意事项

1. **不确定时标注存疑** —— 无法核实的内容标注⚪，不武断判定
2. **提供修改依据** —— 指出错误时说明依据来源
3. **尊重版本差异** —— 古文有不同版本时注明
4. **区分建议与错误** —— 风格优化建议不与事实错误混为一谈
5. **保护原文结构** —— 审查报告清晰区分原文与修改建议
6. **敏感内容谨慎处理** —— 政治、民族、宗教等内容需特别审慎

---

## 限制说明

1. 本技能基于内置知识库，对于极冷门文献可能无法核实
2. 专业领域（如法律、医学）术语需结合专业规范判断
3. 最新时事信息可能存在时效性限制
4. 某些学术争议问题无定论，需注明存在不同观点

### 版本审查流程

```
用户提交含古籍引用的文档
    ↓
提取所有古籍引用（篇目、作者、词句）
    ↓
核对统编教材版本（如适用）
    ↓
如非教材篇目，核对权威校注本
    ↓
发现版本差异 → 标注说明
    ↓
输出版本审查报告
```

### 特别提示

1. **教学用途优先采用统编教材版本** —— 即使与其他权威版本有差异，也应以教材为准
2. **通假字保留原则** —— 教材通常保留通假字原貌（如"说"不改为"悦"）
3. **异体字处理** —— 教材一般采用规范汉字，异体字需转换
4. **标点差异** —— 古籍原无标点，不同版本断句可能有差异，以教材为准
5. **非教材篇目** —— 优先采用中华书局、上海古籍出版社权威校注本
6. **版本注明** —— 学术用途需注明所据版本（如"据中华书局 2020 年版《论语译注》"）

# 中学文言文常用词常用义 编写规范

> 本规范是项目所有字词文档的编写依据。已有文档按规范逐步修订，新增文档严格遵循本规范。

---

## 一、文件规范

### 1.1 文件命名

| 类别 | 格式 | 示例 |
|------|------|------|
| 单字文档 | `字.md` | `之.md`、`为.md`、`乃.md` |
| 双音节词文档 | `词.md` | `其实.md`、`然后.md` |
| 文本合集 | `编号+.md` | `000中学文言文常用词常用义-合集.md`、`001语文教材文言文文本.md`、`002江西省中考语文真题文言文文本.md`、`003古汉语常见双音节词.md` |

- 文件名**不加编号前缀**（编号前缀仅用于合集等汇总文件）
- 纯中文文件名，不含空格、英文、符号
- 不要使用 `~`、`_backup`、`old` 等后缀

### 1.2 文件夹结构

```
古汉语常用字/
├── 000中学文言文常用词常用义-合集.md
├── 001语文教材文言文文本.md
├── 002江西省中考语文真题文言文文本.md
├── 003古汉语常见双音节词.md
├── README.md
├── 编写规范.md
├── 常用词/          ← 所有单字文档，放在此子目录
└── 近义词辨析：……   ← 其他辅助文档（可选）
```

## 二、文档结构

每个字词文档必须包含以下全部内容，按以下顺序排列：

```
# 字 (拼音)

**【词性】** 词性列表（顿号分隔）

**【频率】** 出现N次

---

## 义项1：词性——核心释义

**【释义】** 释义文字。

**【例句】**
1. 《出处·篇名》：原文例句。

---

（后续义项同理）

## 【中考重点】

正文内容。

## 【学习提示】

正文内容。
```

### 2.1 文档头部

```
# 字 (拼音)
```

- 标题：`#` + 半角空格 + 字（不加书名号）+ 半角空格 + 拼音（不加括号）
- 拼音只注一种读音；有文白异读或词性异读的，在相关义项内单独说明，不在标题重复标注
- 标题下方紧跟词性和频率元数据，不空行

### 2.2 元数据行

```
**【词性】** 词性列表

**【频率】** 出现N次
```

- **【词性】**和**【频率】**均为必选项，缺一不可
- 词性使用顿号（、）分隔，顺序按义项出现先后排列
- 频率数字来自教材文本 + 中考真题文本的合计出现次数（自动统计）

## 三、义项格式（核心规范）

### 3.1 标题层级

- **义项标题**：必须使用 `##`，不得使用 `###`
- **可选子节**（【固定搭配】【成语积累】【一词多义比较】等）：使用 `###`

```
## 义项1：动词——做

### 【固定搭配】

### 【成语积累】
```

### 3.2 义项编号与标题格式

- **编号**：阿拉伯数字 `1、2、3……`，禁止使用中文数字（"义项一、义项二"）
- **词性标注**：标题内用 `——` 连接词性与释义关键词，词性用标准术语

正确示例：
```
## 义项1：动词——做
## 义项2：介词——因为
## 义项3：连词——如果
## 义项4：副词——竟然
## 义项5：代词——这、此
```

常见词性术语参考：动词、名词、形容词、副词、代词、介词、连词、助词、语气词、叹词

### 3.3 义项内部结构

每个义项必须包含以下两个部分，**顺序固定，不得颠倒**：

```
## 义项X：词性——释义关键词

**【释义】** 释义文字。

**【例句】**
1. 《出处·篇名》：原文例句。
```

#### 【释义】格式规范

- 释义以简洁准确为目标，主释义在首位
- 多个近义译法用顿号（、）分隔：`相当于"是""成为"`
- 若该义项在教材中出现频率极低（如仅见于生僻文本），可在释义末尾加括注说明：`此义项在初中教材中较少出现。`
- **严禁空释义**：每个义项必须有实质释义，不可仅有"释义文字"字样占位

#### 【例句】格式规范

**3.3.1 基本格式**

每个例句格式为：`《出处》：原文例句。`

| 要素 | 规范 |
|------|------|
| 出处格式 | `《书名·篇名》`（书名与篇名之间用 `·`，不是 `.` 或其他符号） |
| 冒号 | 全角中文冒号 `：` |
| 原文 | 不使用引号，不加书名号，不使用单/双引号 |
| 句末点号 | 句号（。）、问号（？）、感叹号（！）必选其一 |
| 编号 | 阿拉伯数字 + 半角句点（`.`）+ 半角空格：`1. ` |
| 换行 | 每条例句单独一行，相邻例句之间**不空行** |

**3.3.2 例句来源优先级**

1. **首选**：初中语文教材（`001语文教材文言文文本.md`）
2. **次选**：江西省中考语文真题（`002江西省中考语文真题文言文文本.md`，2014—2025年）
3. **补充**：其他可靠来源——经典古籍（经史子集）、权威文献

> **注**：同一义项例句数量原则上不少于1句；高频常用词的主要义项建议配2—3条例句，覆盖不同出处。

**3.3.3 例句夹注**

如果例句中目标字词的意思需要补充说明，可在句末括号内简要注释：

```
1. 《史记·陈涉世家》：今诚以吾众诈自称公子扶苏、项燕，为天下唱，宜多应者。（以：如果；为：为了）
```

括号内注释格式：
- 被注释的字词用全角引号标出：`"以"`
- 注释文字不加引号
- 多条注释之间用分号（；）分隔

**3.3.4 例句中不使用引号**

❌ 错误：
```
1. 《论语》：子曰："学而时习之，不亦说乎？"
2. 《诗经》："关关雎鸠，在河之洲。"
```

✅ 正确：
```
1. 《论语·学而》：子曰：学而时习之，不亦说乎？
2. 《诗经·关雎》：关关雎鸠，在河之洲。
```

> 例句原文一律不加引号包裹，去除原文自带的所有引号，保留其余内容。

## 四、通假字与古今字

### 4.1 「通某字」格式

使用中文全角引号（**""**）：

❌ 错误：
```
## 义项N：通"已"——已经
```

✅ 正确：
```
## 义项N：动词——通"已"，已经
```

### 4.2 「某字的后起字/古字」格式

同样使用中文全角引号：

```
## 义项N：名词——"花"（花）的古字
```

## 五、可选子节

以下子节为可选项，根据需要添加在同一义项释义之后，或文档末尾。子节标题层级为 `###`，格式为 `### 【子节名】`。

### 5.1 【固定搭配】

用于列出该字词的常见双音节词或固定短语（可附简要释义）：

```
### 【固定搭配】

- **自若**：像自己原来的样子，不变常态。
- **自然**：不经人力干预而如此。
```

### 5.2 【成语积累】

用于列出包含该字词的常见成语（可附简要释义）：

```
### 【成语积累】

- 温故知新——温习学过的知识，能得到新的理解。
- 叹息**之**声——叹息的声音。
```

### 5.3 【一词多义比较】

用于对比该字与其他相近字词的异同：

```
### 【一词多义比较】

"之"与"其"：两者均可作代词，但"之"多作近指（这），"其"多作远指（那）。
```

## 六、【中考重点】规范

```
## 【中考重点】

"字"字在文言文中主要考查以下义项：
1. **词性**：释义关键词
2. **词性**：释义关键词
3. ……
```

- 列出该字**最常考查**的义项（一般3—5个），不超过5个
- 使用有序列表（`1. `），粗体词性，释义关键词跟随其后
- 词性术语与义项标题保持一致

---

## 七、【学习提示】规范

```
## 【学习提示】

正文内容。
```

- 提供**辨析、记忆方法、常见陷阱**等帮助性内容
- 可进行同类字词对比（同义词辨析、与现代汉语的区别等）
- 避免空泛的"需要根据语境判断"类套话

## 八、标点符号规范（核心要求）

### 8.1 必须使用中文全角标点

文档中所有标点符号（冒号、顿号、引号、书名号等）一律使用中文全角形式：

| 符号 | 正确 | 错误 |
|------|------|------|
| 冒号 | `：` | `:` |
| 顿号 | `、` | `,` |
| 句号 | `。` | `.` |
| 问号 | `？` | `?` |
| 感叹号 | `！` | `!` |
| 引号 | `""` | `""`、`""`、" " |
| 书名号 | `《》` | `<>` |

### 8.2 例句中不使用引号

**所有例句内容均不得使用任何引号包裹。**

- 例句开头的《书名·篇名》使用书名号，是出处标注，不是引号，**不属于此限制**
- 例句正文去掉原文中自带的引号（如 `"子曰"` → `子曰`）

### 8.3 例句中不使用单引号

同 8.2。例句正文一律不得出现单引号 `'`，无论全角半角。

## 九、合集文件规范

### 9.1 双链格式

合集文件中所有词条统一使用 Obsidian 双向链接格式：

```
[[字]]
```

❌ 错误（旧的 `{{FILE:}}` 格式）：
```
{{FILE:之.md}}
```

✅ 正确：
```
[[之]]
```

### 9.2 双链覆盖要求

合集文件中的**每一个**词条，都必须有对应的独立 `.md` 文件。

- 新增词条时：先创建文档 → 再更新合集添加双链
- 删除词条时：先从合集移除双链 → 再删除文档
- 维护时：定期检查合集双链与文件系统的一致性

## 十、格式自检清单

每篇文档完成后，按以下清单逐项核对：

### 必检项（有一项不符即为不合格）

- [ ] 文件名为单字 + `.md`，无空格无符号
- [ ] 标题格式为 `# 字 (拼音)`，`##` 仅用于义项
- [ ] 义项编号使用阿拉伯数字（`1.`, `2.`），标题内无 `###` 出现
- [ ] **【词性】** 和 **【频率】** 两行元数据完整
- [ ] 每个义项都有 **【释义】**，且释义内容非占位文字
- [ ] 每个义项都有 **【例句】**，至少1句
- [ ] 每条例句格式为 `《出处》：原文。`，末尾有句末点号
- [ ] 例句中**不使用**任何引号（`""`、`''`）
- [ ] 标点符号全部为中文全角，无半角符号混入
- [ ] 文档末尾有 **【中考重点】** 和 **【学习提示】**

### 推荐项

- [ ] 例句来自教材优先，经典文献次之
- [ ] 高频义项配有2—3条例句（不同出处）
- [ ] 义项标题格式：`## 义项X：词性——释义关键词`
- [ ] 文档末尾有【固定搭配】【成语积累】等子节（如适用）

## 附录 A：常见问题处理

### Q1：某个义项找不到例句怎么办？

- 首先扩大搜索范围：教材 → 江西省中考真题（2014—2025）→ 经典古籍
- 若确实无法找到来自教材或中考的例句，可使用其他可靠来源（经史子集），但需在例句后括注说明来源可靠性
- 若某义项仅见于现代白话文或口语，完全不见于文言语境，**考虑删除该义项**，不必强行保留

### Q2：同一个字的不同义项，例句可以相同吗？

- 同一例句中若该字同时体现多个义项（如"自"既有"从"的意思又有"自己"的意思），应将例句放在**最贴切的那个义项**下
- 其他义项不应重复使用同一例句
- 如某例句确实同时体现多个重要义项，可在其他义项下注明"参见义项X"

### Q3：多音字如何处理？

- 标题只注一种读音（最常见或最符合教学目标的读音）
- 不同读音对应不同义项时，在义项标题或释义中注明读音：`## 义项1：动词（fū）——丈夫`

### Q4：词性分类与义项划分的关系？

- 原则上同一个词性下的不同用法，应拆分为不同义项（即使译法相近）
- 例如"之"的助词用法应分为"的"和"取消句子独立性"两个义项，不应合并

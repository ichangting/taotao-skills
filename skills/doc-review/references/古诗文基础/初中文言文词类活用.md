# 初中文言文词类活用

## 名词用作动词

| 字词  | 例句               | 词义             |
| --- | ---------------- | -------------- |
| 前   | 狼不敢前。（《狼》）       | 上前。            |
| 洞   | 一狼洞其中。（《狼》）      | 挖洞。            |
| 蔓、枝 | 不蔓不枝。（《爱莲说》）     | 蔓,横生藤蔓;枝,旁生枝茎。 |
| 糁   | 石青糁之。（《核舟记》）     | 用颜料等涂上。        |
| 腰   | 腰白玉之环。（《送东阳马生序》） | 在腰间佩戴。         |
| 东西  | 任意东西。（《与朱元思书》）   | 向东或向西。         |
| 福   | 神弗福也。（《曹刿论战》）    | 赐福,保佑。         |
| 服   | 朝服衣冠。（《邹忌讽齐王纳谏》） | 穿戴。            |
| 策   | 策之不以其道。（《马说》）    | 用马鞭驱赶。         |
| 军   | 军霸上。（《周亚夫军细柳》）   | 驻军。            |
| 王   | 大楚兴,陈胜王。（《陈涉世家》） | 称王。            |
| 志   | 非志无以成学。（《诫子书》）   | 立志。            |
|     |                  |                |

## 名词用作状语

|字词|例句|词义|
|---|---|---|
|斗、蛇|斗折蛇行。（《小石潭记》）|斗,像北斗星那样;蛇,像蛇那样。|
|先、后|先天下之忧而忧,后天下之乐而乐。（《岳阳楼记》）|先,在······之前;后,在······之后。|
|隧|意将隧入以攻其后也。（《狼》）|从通道。|
|翼|有亭翼然临于泉上者。（《醉翁亭记》）|像鸟张开翅膀。|
|芥|与余舟一芥。（《湖心亭看雪》）|像小草一样微小。|

## 形容词用作名词

|字词|例句|词义|
|---|---|---|
|故、新|温故而知新。（《＜论语＞十二章》）|故,学过的知识;新,新的理解和体会。|
|湍|则素湍绿潭。（《三峡》）|急流。|
|怪|志怪者也。（《北冥有鱼》）|怪异的事物。|
|坚、锐|将军身披坚执锐。（《陈涉世家》）|坚,指铠甲;锐,指武器。|
|良实|此皆良实。（《出师表》）|忠良诚实的人。|
|险|吾与汝毕力平险。（《愚公移山》）|险峻的大山。|
|远|非宁静无以致远。（《诫子书》）|远大目标。|
|老、壮、幼|使老有所终,壮有所用,幼有所长。（《大道之行也》）|老,老年人;壮,中年人;幼,幼童。|

|字词|例句|词义|
|---|---|---|
|轩、邈|互相轩邈。（《与朱元思书》）|轩,往高处伸展;邈,往远处伸展。|
|颓|夕日欲颓。（《答谢中书书》）|坠落。|
|威|威天下不以兵革之利。（《得道多助,失道寡助》）|震慑。|
|私|吾妻之美我者,私我也。（《邹忌讽齐王纳谏》）|偏爱。|
|亲、远|亲贤臣,远小人。（《出师表》）|亲,亲近;远,疏远。|

## 动词的使动用法

|字词|例句|词义|
|---|---|---|
|动|所以动心忍性,曾益其所不能。（《生于忧患,死于安乐》）|使······受到震撼。|
|闻|闻寡人之耳者。（《邹忌讽齐王纳谏》）|使······听到。|
|淫|富贵不能淫。（《富贵不能淫》）|使······惑乱,使······迷惑。|
|移|贫贱不能移。（《富贵不能淫》）|使······改变,使······动摇。|

## 形容词的使动用法

|字词|例句|词义|
|---|---|---|
|乱、劳|无丝竹之乱耳,无案牍之劳形。（《陋室铭》）|乱,使······扰乱;劳,使······劳累。|
|凄、寒|凄神寒骨。（《小石潭记》）|凄,使······感到悲伤;寒,使······感到寒冷。|

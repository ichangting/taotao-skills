# 初中文言文通假字汇编

| 字词  | 例句                        | 词义                          |
| --- | ------------------------- | --------------------------- |
| 有   | 吾十有五而志于学。（《＜论语＞十二章》）      | "有"同"又",用于整数和零数之间。          |
| 说   | 不亦说乎？（《＜论语＞十二章》）          | "说"同"悦",愉快。                 |
| 不   | 尊君在不？（《陈太丘与友期行》）          | "不"同"否"。                    |
| 舍   | 晓之者亦舍然大喜。（《杞人忧天》）         | "舍"同"释",解除、消除。              |
| 尔   | 但手熟尔。（《卖油翁》）              | "尔"同"耳",相当于"罢了"。            |
| 畔   | 亲戚畔之。（《得道多助,失道寡助》）        | "畔"同"叛",背叛。                 |
| 女   | 往之女家。（《富贵不能淫》）            | "女"同"汝",你。                  |
| 要   | 便要还家。（《桃花源记》）             | "要"同"邀",邀请。                 |
| 阙   | 略无阙处。（《三峡》）               | "阙"同"缺",空隙、缺口。              |
| 转   | 蝉则千转不穷。（《与朱元思书》）          | "转"同"啭",鸟鸣,这里指蝉鸣。           |
| 反   | 窥谷忘反。（《与朱元思书》）            | "反"同"返",返回。                 |
| 食   | 食马者不知其能千里而食也。（《马说》）       | "食"同"饲",喂。                  |
| 见   | 才美不外见。（《马说》）              | "见"同"现"。                    |
| 支   | 四支僵劲不能动。（《送东阳马生序》）        | "支"同"肢"。                    |
| 被   | 同舍生皆被绮绣。（《送东阳马生序》）        | "被"同"披"。                    |
| 具   | 百废具兴。（《岳阳楼记》）             | "具"同"俱",全、皆。                |
| 属   | 属予作文以记之。（《岳阳楼记》）          | "属"同"嘱",嘱托。                 |
| 式   | 改容式车。（《周亚夫军细柳》）           | "式"同"轼",车前横木。这里是名词用作动词,指扶轼。 |
| 衡   | 左手倚一衡木。（《核舟记》）            | "衡"同"横"。                    |
| 与   | 选贤与能。（《大道之行也》）            | "与"同"举"。                    |
| 矜   | 矜、寡、孤、独、废疾者皆有所养。（《大道之行也》） | "矜"同"鳏",老而无妻。               |
| 曾   | 曾益其所不能。（《生于忧患,死于安乐》）      | "曾"同"增"。                    |
| 衡   | 衡于虑。（《生于忧患,死于安乐》）         | "衡"同"横",梗塞、不顺。              |
| 拂   | 入则无法家拂士。（《生于忧患,死于安乐》）     | "拂"同"弼",辅佐。                 |
| 辟   | 故患有所不辟也。（《鱼我所欲也》）         | "辟"同"避",躲避。                 |
| 辩   | 万钟则不辩礼义而受之。（《鱼我所欲也》）      | "辩"同"辨",辨别。                 |
| 得   | 所识穷乏者得我与？（《鱼我所欲也》）        | "得"同"德",感恩、感激。              |
| 以   | 固以怪之矣。（《陈涉世家》）            | "以"同"已"。                    |
| 唱   | 为天下唱。（《陈涉世家》）             | "唱"同"倡",倡导、发起。              |
| 错   | 故不错意也。（《唐雎不辱使命》）          | "错"同"措"。                    |
| 反   | 始一反焉。（《愚公移山》）             | "反"同"返",往返。                 |
| 惠   | 汝之不惠。（《愚公移山》）             | "惠"同"慧",聪明。                 |

# 板书设计参考

> 本文件提供常见板书布局类型与适用场景，供 kejian-generator 技能使用。
> 板书是课堂的"视觉锚点"，好的板书让学生一眼抓住这节课的骨架。

---

## 一、主板书布局类型

### 1. 左右分栏式

```
┌─────────────────────────────────────┐
│  左栏（文本/线索）  │  右栏（分析/对比）  │
│                     │                     │
│  段落/情节梳理      │  手法/情感/主题     │
│  时间线/事件线      │  人物/环境/冲突     │
└─────────────────────────────────────┘
```

适用：叙事类课文（记叙文、小说）、对比阅读、文言文"原文-翻译"对照。

### 2. 上下分层式

```
┌─────────────────────────────────────┐
│  上层：核心问题 / 主旨               │
├─────────────────────────────────────┤
│  中层：分层展开（段意/层次/论据）     │
├─────────────────────────────────────┤
│  下层：总结 / 关键词 / 方法提炼       │
└─────────────────────────────────────┘
```

适用：议论文、说明文、需要"总-分-总"结构呈现的课文。

### 3. 中心辐射式

```
        线索A
          ↖
  线索B ← 核心 → 线索C
          ↙
        线索D
```

适用：散文（形散神聚）、诗歌鉴赏（意象围绕情感辐射）、名著导读（人物关系）。

### 4. 时间轴/流程式

```
起因 ──→ 经过 ──→ 高潮 ──→ 结局
 ↓        ↓        ↓        ↓
(关键词) (关键词) (关键词) (关键词)
```

适用：叙事性强的课文、历史事件、实验/说明流程。

### 5. 对比式

```
┌────────────┬────────────┐
│   对象 A   │   对象 B   │
├────────────┼────────────┤
│  特征 1    │  特征 1    │
│  特征 2    │  特征 2    │
│  特征 3    │  特征 3    │
├────────────┴────────────┤
│     共同点 / 差异点      │
└─────────────────────────┘
```

适用：对比阅读、人物对比、古今异义对比、两首诗对比鉴赏。

### 6. 阶梯递进式

```
                    主旨/升华
                  ╱
            深层理解
          ╱
    表层含义
  ╱
字词/基础
```

适用：古诗文鉴赏（字词→句意→手法→情感→主旨）、议论文论证层次。

## 二、副板书使用规范

| 项目 | 建议 |
|------|------|
| 位置 | 主板书右侧或下方，与主板书有明确视觉分隔 |
| 用途 | 学生发言关键词、临时补充、易错字/音、课堂生成性资源 |
| 书写 | 可潦草，不必如主板书工整 |
| 保留 | 随写随擦（保持整洁）或保留到下课（供学生抄录），视教学需要 |
| 容量 | 不超过黑板面积的 1/3 |

---

## 三、板书设计原则

1. **少而精**：主板书不超过 30 个字（关键词/短语），不写完整句子。
2. **结构化**：用箭头、括号、线条表达逻辑关系，不堆砌文字。
3. **同步性**：板书随教学进度逐步呈现，不一次性写满（保留悬念与节奏）。
4. **可复制**：学生能照着板书复述课堂内容（检验板书是否清晰）。
5. **与课件互补**：板书承载"结构骨架"，课件承载"细节与素材"；两者不重复。
6. **书写规范**：教师板书本身是示范——字要端正，笔顺要正确（尤其文言文字词）。

## 四、不同课型的板书建议

| 课型 | 推荐布局 | 主板书核心内容 |
|------|----------|----------------|
| 现代文阅读（记叙文） | 左右分栏 / 时间轴 | 情节线 + 情感线 |
| 现代文阅读（议论文） | 上下分层 / 阶梯递进 | 论点 + 论据 + 论证方法 |
| 古诗文教学 | 阶梯递进 / 左右分栏 | 字词→句意→手法→情感 |
| 古诗鉴赏 | 中心辐射 | 情感核心 + 意象辐射 |
| 写作指导课 | 上下分层 / 对比式 | 写作要点 + 示例对比 |
| 名著导读 | 中心辐射 / 时间轴 | 人物关系 / 情节主线 |
| 复习课 | 对比式 / 上下分层 | 知识网络 + 易错点 |

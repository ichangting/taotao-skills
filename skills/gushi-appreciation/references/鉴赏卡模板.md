# 鉴赏卡模板（输出格式锁死）

> 本文件定义 `gushi-appreciation` 单篇鉴赏卡的**唯一输出格式**。生成鉴赏卡时严格套用，不得省略必填栏。
> 古文引用（作者/篇目/原文）输出前须经 `doc-review` 古文引用专项收口；默写易错字/通假字回查 `guwen-cidian`。

---

## 必填栏（顺序不可调）

```
【古诗文鉴赏卡】<篇目> · <年级/册次> · <诗/词/曲/文>

一、原文与出处
- 原文：<全文，逐句核对作者与篇目无张冠李戴>
- 作者：<姓名·朝代>  出处：<集名/教材篇目>
- 〔经 doc-review 古文专项收口：作者/篇目/原文无误〕

二、主旨情感（标注定评/有争议）
- 主流定评：<1–2 句点明核心情感与主旨>
- 争议提示：<若无争议写"学界基本一致">；若有，写明分歧点

三、意象分析（逐意象，带原句证据）
- <意象1>（原句："……"）：常规含义 <X>，本篇具体指向 <Y>
- <意象2>（原句："……"）：……
（无意象可不列，古文篇目常偏手法而非意象）

四、艺术手法（逐处点明 + 判定线索，引原句为证）
- <手法1>：原句"……"，判定线索 <为何判为此手法>
- <手法2>：……
（参考 references/手法库.md，不堆术语，多重手法须分别列出）

五、名句赏析（选 1–2 句，三层拆解）
- 原句："……"
  · 字面：<句意>
  · 手法：<用了什么手法>
  · 情感：<传达何种情感/哲理>

六、背诵默写易错字（回查 guwen-cidian）
- <字>：易错写法是 <错>，正写 <对>；<是否通假，如"属"通"嘱">
- （回查 references/古诗文基础/中考语文古诗文默写易错字.md 与 通假字.md）

七、中考链接（可选，回查考情数据）
- 若已安装 jiangxi-zhongkao-beikao / wordfreq：列出该篇近年考查频次与题型
- 未安装：基于通用经验提示常见题型，并标注"以当年真题为准"

八、定评 vs 商榷（必须分两栏，商榷须真有讨论空间）
- 主流定评：
  · <学界/教材教参普遍采纳的说法>
- 可讨论点：
  · <存在多种合理解读或学界争议处，如作者归属/情感基调/某句解读>
  · （不得把"可讨论点"写成定评的补充；要真有可展开空间）
```

---

## 格式纪律

1. 顶部统一 `【古诗文鉴赏卡】` 前缀，含篇目 + 年级册次 + 体裁。
2. 每条手法/意象判定**必须引原句为证**，禁止空发议论（"这首诗运用了借景抒情"式无效）。
3. "定评 vs 商榷"两栏独立；商榷栏不得为空话，须指明确实存在的解读分歧。
4. 不生成"满分鉴赏模板"让学生背诵套用；不替学生下鉴赏题结论。
5. 古文引用收口标记必须出现（作者/篇目/原文无误），否则视为未收口。

## 反例（错误示范，禁止）

```
❌ 【鉴赏卡】天净沙
手法：借景抒情、托物言志、对比。（无原句、无判定线索、乱贴标签）
定评：表达了作者的思乡之情。商榷：无。（商榷栏空话）
```

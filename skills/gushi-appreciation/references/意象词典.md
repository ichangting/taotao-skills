# 意象词典（初中古诗文高频意象 · 常规含义与反例）

> 用途：教师讲解"某意象象征什么"时查此表。每条给 **常规含义 + 反例/注意 + 代表诗句**。
> 重要：意象是**常规倾向**不是铁律。务必结合具体诗境，避免学生套模板误读（见各条"反例"）。

---

## 自然类

### 月（月亮 / 明月 / 月圆）
- **常规含义**：思乡怀人、团圆期盼、时空永恒（对照人生短暂）。
- **代表**：李白"举头望明月，低头思故乡"；苏轼"但愿人长久，千里共婵娟"；张九龄"海上生明月，天涯共此时"。
- **反例/注意**：也可写清冷孤寂（"月落乌啼霜满天"）或壮景（"明月别枝惊鹊"），非必哀。

### 流水
- **常规含义**：时光流逝、愁绪绵长（"问君能有几多愁，恰似一江春水向东流"）。
- **反例**：也可单纯写景（"清流见底"），需看是否带情感。

### 夕阳 / 落日 / 昏鸦
- **常规含义**：迟暮、思归、衰飒、羁旅之愁。
- **代表**："夕阳西下，断肠人在天涯"；"大漠孤烟直，长河落日圆"（此处落日偏壮美，注意反例）。

### 春风 / 东风
- **常规含义**：希望、生机、皇恩（"春风又绿江南岸"）。
- **反例**："东风不与周郎便"中东风指机遇/天时，非自然春意。

### 落花 / 落红 / 残红
- **常规含义**：伤春惜时、美好消逝、身世飘零。
- **代表**："无可奈何花落去"；"流水落花春去也"。
- **反例/升华**：龚自珍"落红不是无情物，化作春泥更护花"——落花转为奉献，是逆用。

## 植物类

### 柳（杨柳 / 折柳）
- **常规含义**：惜别、留恋（"柳"谐"留"；折柳送别习俗）。
- **代表**："昔我往矣，杨柳依依"；"客舍青青柳色新"。

### 雁（大雁 / 归雁）
- **常规含义**：思乡传书、漂泊羁旅（雁足传书典）。
- **代表**："乡书何处达？归雁洛阳边"；"塞下秋来风景异，衡阳雁去无留意"。

### 菊
- **常规含义**：高洁、隐逸、傲霜（陶渊明菊文化）。
- **代表**："采菊东篱下，悠然见南山"。

### 梅
- **常规含义**：坚贞、孤高、报春（凌寒独放）。
- **代表**：陆游《卜算子·咏梅》"零落成泥碾作尘，只有香如故"（反衬遭际而志不变）。

### 竹
- **常规含义**：气节、虚心、坚韧。

### 莲（荷）
- **常规含义**：高洁（"出淤泥而不染"）；"莲"谐"怜"（爱），见于南朝乐府。

### 梧桐
- **常规含义**：凄凉、离情（梧桐更兼细雨）；也指高洁（凤凰非梧桐不栖）。

## 动物类

### 杜鹃（子规 / 杜宇）
- **常规含义**：悲苦、思归、哀怨（啼血典）。
- **代表**："杨花落尽子规啼"；"又闻子规啼夜月"。

### 鹧鸪
- **常规含义**：离愁别绪、羁旅之悲（啼声似"行不得也哥哥"）。
- **代表**："江晚正愁余，山深闻鹧鸪"。

### 蝉
- **常规含义**：高洁（居高声自远）、悲秋、生命短暂。

### 孤雁 / 孤鸿
- **常规含义**：孤寂、失群、飘零（"缥缈孤鸿影"）。

---

## 人文类

### 酒
- **常规含义**：愁绪（"酒入愁肠"）、豪情（"葡萄美酒夜光杯"）、送别（"劝君更尽一杯酒"）。
- **注意**：酒在诗中功能多，需看语境定是"借酒浇愁"还是"壮行助兴"。

### 舟 / 帆
- **常规含义**：漂泊（"孤舟蓑笠翁"）、离别（"孤帆远影碧空尽"）、自由（"轻舟已过万重山"）。

### 长亭 / 南浦 / 阳关
- **常规含义**：送别之地（"长亭更短亭""西出阳关无故人"）。

### 羌笛 / 胡笳 / 戍鼓
- **常规含义**：边塞征戍之悲（"羌笛何须怨杨柳""戍鼓断人行"）。

---

> 教学铁律：**意象含义随诗境而变**。讲意象时必带"在本诗中具体指向什么"，并提示反例，防止学生背"月=思乡"到处套。

# 古诗文鉴赏（gushi-appreciation）

## 目的

为一线语文教师提供**单篇古诗文**的鉴赏教学支持——从"字词"（guwen-cidian 已解决）往上走到**意象、手法、情感、考法**，帮教师快速组织赏析环节，且**始终区分"主流定评"与"可讨论点"**。

核心理念：**语文鉴赏本就多元，本技能不产出"唯一标准答案版"鉴赏**。它给教师的是结构化素材 + 可展开的争议点，最终的教学判断权在教师、在课堂。

## 何时使用

- "《天净沙·秋思》怎么赏析""帮我分析这首诗的意象"
- "这首诗用了什么艺术手法""借景抒情还是托物言志怎么区分"
- "这首诗的主旨情感是什么""名句赏析怎么讲"
- "这篇在江西/全国中考怎么考""默写易错字有哪些"（易错字回查 guwen）
- 备课、公开课、整本书阅读里的古诗文专题、中考复习里的鉴赏题型梳理。

## 与 guwen-cidian 的分工（单一数据源）

## 核心能力（MVP，四种用法）

### 用法一：单篇鉴赏卡（主场景）

输入一篇古诗文（诗/词/曲/古文），输出一张结构化鉴赏卡。格式**锁死**见 `references/鉴赏卡模板.md`，要点：

1. **原文 + 出处 + 作者**（作者/篇目/原文先过 doc-review 古文专项收口）。
2. **主旨情感**：用 1–2 句点明，标注"主流定评"或"学界有争议"。
3. **意象分析**：逐意象说明常规含义 + 在本篇的具体指向（参考 `references/意象词典.md`）。
4. **艺术手法**：列出手法并给判定线索（参考 `references/手法库.md`），不堆术语。
5. **名句赏析**：选 1–2 句做"字面 + 手法 + 情感"三层拆解。
6. **背诵默写易错字**：回查 guwen-cidian 默写易错字表 + 通假字表，逐条标注。
7. **中考链接**（可选）：回查 `jiangxi-zhongkao-beikao` / `wordfreq`（若已安装）；未安装则基于通用经验提示并标注"以当年真题为准"。
8. **定评 vs 商榷**：单独一节，列出"学界主流定评"与"教学可展开的讨论点"。

### 用法二：意象词典查询

教师问"月/柳/雁 在古诗里一般象征什么"，查 `references/意象词典.md` 给常规含义 + 反例（避免学生套模板误读）。

### 用法三：手法判定辅助

教师拿不准"这是借景抒情还是托物言志""虚实结合怎么判断"，查 `references/手法库.md` 的"判定线索"栏，给可操作标准而非死记定义。

### 用法四：考法梳理

针对某篇，梳理近年中考常见鉴赏题型（意象题/手法题/情感题/名句理解题/对比阅读题），回查考情数据（可选）。

## 输出格式（锁死）

严格按 `references/鉴赏卡模板.md` 渲染。关键约束：

- 鉴赏卡顶部标注 `【古诗文鉴赏卡】<篇目> · <年级/册次>`。
- 每一条"手法/意象"判定都要带**文本证据**（引原句），不空发议论。
- "定评 vs 商榷"必须分两栏，商榷栏不得写成"定评的补充"，要真有可讨论空间（如作者归属、情感基调、某句解读分歧）。

## 定评 vs 商榷 原则（红线）

- **主流定评**：学界/教材教参普遍采纳的说法，可放心用于教学。
- **可讨论点**：存在多种合理解读、或学界有争议处（如《诗经·蒹葭》"伊人"所指、李清照前后期词风分界的具体作品归属、某诗是否确为"借古讽今"存疑等）。
- ❌ 不把"可讨论点"包装成唯一答案；❌ 不替学生下鉴赏题结论；❌ 不生成"满分鉴赏模板"让学生背诵套用。

## 诚信与合规护栏

- **以教材/教参为基线**：主旨与手法解读优先采用统编教材与主流教参；若存争议，明确标注"学界有不同看法，可课堂展开"，不臆造定论。
- **古文引用收口**：作者、篇目、原文在输出前经 `doc-review` 古文引用专项核对，避免张冠李戴（如把《岳阳楼记》作者写成李白）。
- **不向学生收费 / 不替代教师**：本技能定位教师免费公共资产；输出供教师备课参考，判定权与教学责任在教师。
- **品牌声音**：讲解语气参考 [品牌声音与品牌规范](../../docs/品牌声音与品牌规范.md)（准确、干净、不浮夸、不堆术语）。

## 新增语料规范

- 意象词典新增意象：套用 `references/意象词典.md` 现有写法（常规含义 + 反例 + 代表诗句）。
- 手法库新增术语：补"定义 + 判定线索 + 初中代表篇目"三栏。
- 字词/通假/默写类内容：**回 guwen-cidian 增补**，本技能不另存。

## 参考资料（references/）

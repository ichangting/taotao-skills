# 整本书阅读与名著导读体系化设计技能

## 技能定位

本技能专为语文教师设计，**以《义务教育语文课程标准（2022 年版）》为依据**，提供整本书阅读与名著导读的全流程闭环教学方案，解决"放羊式阅读、只考不教"的痛点。

---

## 课标依据

### 学习任务群目标

> 本学习任务群旨在引导学生在语文实践活动中，根据阅读目的和兴趣选择合适的图书，制订阅读计划，综合运用多种方法阅读整本书；借助多种方式分享阅读心得，交流研讨阅读中的问题，积累整本书阅读经验，养成良好阅读习惯，提高整体认知能力，丰富精神世界。

### 第四学段（7-9 年级）学习内容

| 类别 | 内容 | 课标要求 |
|------|------|----------|
| 革命文学作品 | 《革命烈士诗抄》《红岩》《红星照耀中国》 | 体会、评析革命领袖、革命英雄的爱国精神和人格魅力 |
| 文学名著 | 《朝花夕拾》《骆驼祥子》《艾青诗选》《西游记》《格列佛游记》《钢铁是怎样炼成的》 | 根据阅读进度完成读书笔记，针对语言、形象、主题等话题展开研讨 |
| 读书活动 | 多样的读书活动 | 讲述推荐名著、改编精彩片段、撰写文学鉴赏文章 |

### 课标教学提示

1. **统筹安排**：课内与课外、个人与集体相结合，保证课堂阅读时间
2. **自主阅读为主**：引导学生运用浏览、略读、精读等方法，重视序言、目录作用
3. **多样活动**：师生共读、同伴共读、朗诵会、故事会、戏剧节、读书共同体
4. **过程评价**：以阅读态度、方法、笔记为依据，编制评价量表、阅读反思单

## 核心功能（对标课标）

### 1. 阅读计划制定 ✅ 课标要求

| 模块 | 内容 |
|------|------|
| 阅读目的设定 | 根据课标要求和学生实际确定阅读目的 |
| 图书选择指导 | 选择适宜的版本，认识不同类型图书特点 |
| 时间规划 | 合理规划阅读时间，集中使用整本书阅读课时 |
| 进度安排 | 通读整本书，了解主要内容 |

### 2. 阅读方法指导 ✅ 课标要求

| 方法 | 适用场景 | 指导要点 |
|------|----------|----------|
| 浏览 | 初读、了解大意 | 关注序言、目录、章节标题 |
| 略读 | 快速把握内容 | 抓主要情节、关键人物 |
| 精读 | 深入研读 | 品味语言、分析形象、探究主题 |
| 批注 | 全程使用 | 记录疑问、感悟、精彩语句 |

### 3. 读书笔记设计 ✅ 课标要求

| 类型 | 内容 | 适用阶段 |
|------|------|----------|

| 人物分析 | 性格特点、形象意义 | 研读阶段 |
| 语言赏析 | 精彩语句、写作手法 | 精读阶段 |
| 主题探究 | 主旨思想、现实意义 | 深化阶段 |
| 阅读反思 | 阅读方法、心得体会 | 全程 |

### 4. 读书活动设计 ✅ 课标要求

## 教学流程设计（对标课标）

### 阶段一：读前准备（1 课时）

| 环节 | 活动 | 课标对应 |
|------|------|----------|
| 激趣导入 | 影视片段、作者故事、时代背景 | 激发阅读兴趣 |
| 方法指导 | 浏览、略读、精读方法讲解 | 阅读方法指导 |
| 计划发布 | 阅读计划、进度安排、评价标准 | 制订阅读计划 |
| 版本选择 | 推荐适宜版本，说明选择理由 | 选择图书和版本 |

### 阶段二：读中推进（2-4 课时）

| 阶段 | 任务 | 课标对应 |
|------|------|----------|
| 初期（1/3） | 通读全书，梳理情节，完成读书笔记 | 通读整本书，了解主要内容 |
| 中期（2/3） | 精读重点章节，探究语言、形象、主题 | 针对语言、形象、主题展开研讨 |
| 后期（3/3） | 深化理解，拓展延伸，准备展示 | 撰写文学鉴赏文章 |

### 阶段三：读后展示（2-3 课时）

| 活动 | 形式 | 课标对应 |
|------|------|----------|
| 成果展示 | 读书报告、思维导图、手抄报 | 分享阅读心得 |
| 改编表演 | 课本剧、片段表演 | 尝试改编名著片段 |
| 交流研讨 | 主题辩论、问题研讨 | 交流研讨阅读中的问题 |
| 反思总结 | 阅读反思单、自我评价 | 自我反思、自我改进 |

## 课标教学建议落实

### 1. 立足核心素养，彰显育人导向 ✅

| 核心素养维度 | 整本书阅读中的落实 |
|--------------|------------------|
| **文化自信** | 传承三种文化（优秀传统文化、革命文化、先进文化） |
| **语言运用** | 在阅读中积累语言，提升表达能力 |
| **思维能力** | 分析人物、情节、主题，培养批判性思维 |
| **审美创造** | 鉴赏文学语言，创作读书成果 |

**育人目标**：
- 《红岩》《红星照耀中国》→ 革命文化教育，爱国主义精神
- 《西游记》《水浒传》→ 传统文化教育，民族精神
- 《简·爱》《钢铁是怎样炼成的》→ 价值观教育，人格塑造

### 2. 体现学习任务群特点 ✅

**所属任务群**：整本书阅读

| 任务群要求 | 技能设计对应 |
|------------|--------------|
| 通读整本书，了解主要内容 | 读中推进任务单（情节梳理） |
| 关注整体与局部关系 | 分阶段阅读设计 |
| 重视序言、目录作用 | 读前指导包含版本介绍 |

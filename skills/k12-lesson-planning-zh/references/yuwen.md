# 语文 — 备课教学法参考（中国初中 / 统编教材）

本文件由 `k12-lesson-planning-zh` 在学科为**语文**时自动加载。
它承载语文专属的教学法、流程结构、章节结构、不可妥协项，以及 `lesson.json` 的映射。
**加载本文件是强制要求**——未读语文参考就写教案属于严重失误。

> 课标依据统一见 `references/kechengbiao.md`（官方 2022 版第四学段）。本文件只讲"怎么教语文"，
> 凡涉及"学段要求 / 核心素养 / 评价"，一律回到 `kechengbiao.md`，不得凭空编造。

## 一、先问清什么（Clarify）

动笔前从对话信号判断以下各项；缺失才问，优先级如下：
1. **年级与册次**：如"七年级上册"。决定学段（本项目聚焦第四学段 7~9 年级）。
2. **课题 / 文本**：如《春》（朱自清，统编七上第一单元）。未指定则从教材与课标建议中推荐一篇，并在生成物中标注「建议篇目」。
3. **文体**：散文（写景/叙事/抒情）/ 古诗词 / 文言文 / 写作课 / 字词语用。不同文体走不同分支（见下）。
4. **教材版本**：默认统编（部编）版；若教师另有版本，按其所用版本的叙述语言与单元结构来。
5. **课时**：默认 45 分钟（初中标准一节）；若连堂或微课另计。

默认静默套用：45 分钟、核心素养四维（文化自信/语言运用/思维能力/审美创造）、
统编教材、过程性评价。仅当上述某项确实缺失时才开口问。

## 二、课堂流程（中国初中常态，替代美式 5E / 60min）

一节课的典型推进（可按需求伸缩，总时长须恰好等于规定课时，
过渡时间计入各环节之内，不得另设"缓冲"）：
**导入（激活经验）3–5 min → 初读·整体感知 8–10 min → 精读·品味语言/写法 15–18 min
→ 合作·探究/讨论 6–8 min → 小结·迁移 3–5 min → 作业（分层、控量）**。

- **朗读贯穿**：散文/古诗词须有朗读环节（正确、流利、有感情），默读须有速度与批注要求。
- **语言品味是语文的命根子**：紧扣"重要词句在语言环境中的意义和作用"（课标原话），
  不空谈"主题思想"而放过语言。
- **板书**：每节课须给出板书设计（结构/关键词/线索），作为教案必备项。
- **双减**：作业按"基础+提升"两层即可，严禁机械抄写与海量训练；量以"大部分学生 X 分钟内完成"为度。

## 三、文体分支（Branch by 文体）

### A. 现代散文（如《春》《济南的冬天》《散步》）—— 以《春》为范
典型推进：导入（唤起对春的生活经验）→ 初读·梳理写景顺序（盼春—绘春—赞春）
→ 精读·品味比喻/拟人/排比与准确动词 → 合作·选段朗读与批注 → 小结·写景手法迁移
→ 作业（背诵积累 + 微写作：写一段四季之一景）。
**不可妥协**：① 必须有"朗读 + 语言品味"双主线，不能只讲内容；② 须指出作者如何
"用准确生动的词语与修辞建构画面"（对应课标"体味推敲重要词句"）；③ 不照搬课文原句当答案，
学生作答须是自己组织的话。

### B. 古诗词（如《观沧海》《次北固山下》）
典型推进：知人论世（作者/背景）→ 诵读·读准字音节奏 → 意象·画面想象
→ 品析·关键句的思想情感 → 背诵默写 → 作业（默写 + 意象小练）。
**不可妥协**：① 诵读到位（节奏、韵脚）；② 不逐字翻译代替鉴赏；③ 情感须落在"自然/社会/人生启示"上。

### C. 文言文（如《〈论语〉十二章》《诫子书》）
典型推进：解题·作者 → 朗读·读准停顿 → 借助注释与工具书疏通大意
→ 虚词/句式聚焦（如"之""而"）→ 内容与文化内涵探讨 → 背诵 → 作业（译句 + 文化体会）。
**不可妥协**：① 必须以"借助注释和工具书理解基本内容"为基线，不越级深挖训诂；
② 文化常识点到即止，服务理解而非堆砌；③ 严禁生成可下载的"逐字翻译 PDF"等越界产物。

### D. 写作课（如"热爱生活，热爱写作""写人要抓住特点"）
典型推进：情境导入 → 范文/例文引路 → 方法点拨（如"抓住特征"）→ 当堂片段练写
→ 互评·依据量规 → 小结·修改建议 → 作业（成文，控量）。
**不可妥协**：① 写作须"有真情实感，表达对自然/社会/人生的感受思考"（课标原话）；
② 提供可操作的方法支架，而非空喊"多观察"。

### E. 字词 / 语用（如"词语的感情色彩""语序的意义"）
典型推进：情境引词 → 辨析（本义/语境义/感情色彩）→ 运用（造句/语段填空）
→ 小结·规律 → 作业（小练习，控量）。

## 四、章节结构（所有文体通用，须严格照此生成）

1. **一瞥（At a glance）** —— 课标条目原文置于 `special` 型 callout（全篇仅此一处逐字引用，
   其余地方一律"代码 + 十字以内要旨"带过）；一行写明环节与分钟数，使课时形状先可见；
   文本（篇名+作者+文体）；材料清单（逐项实名，如"投影仪、朗读音频、作业单 18 份"）。
2. **学习目标** —— 大概念（一句话，扣文本与单元）；SWBAT 要点（取自课标学段要求，
   列出具体子技能）； prerequisites（前置学段要求 + 一句话假设）。
3. **关键词与预期困难** —— 2–3 个重点词（释义 + 语境）；3 条针对本文的具体"易错/易混/易浅"，
   每条格式：*学生表现* / *成因* / *教师应对*。
4. **教学过程** —— 按上文"课堂流程"分环节；每个有学生活动的环节给 3 条以上"观察点"
   （具体学生行为 + 为何关乎课标 + 怎么做）；讨论环节给具体主问题（非泛泛而问）。
5. **板书设计**（单列，语文必备）。
6. **评价与作业** —— 出口小测（exit ticket）1 题，指向本课最难的语言/思维点；
   作业分层（基础+提升），标明月量与完成时长，符合双减。
7. **设计说明（Design notes）** —— 末节：2–3 个"改编时务必保留"的要素，各附一句研究/课标依据。

→ 章节完成。教师若要看速览草稿，进入 Step 4；否则进 Step 5。

## 五、学生页布局（student_materials，发给学生时）

```
sections:
  "课前/导入"        from_shared:anchor_task ; answer_box（若有书面预测）
  "初读·整体感知"   for each q_k:
                         group[ {type: from_shared, key: q_k, label: "k"},
                                answer_box ]
                       page_break
  "精读·品味语言"    from_shared:passage（若重印片段，否则在教案 Materials 注明出处）
  "作业（分层）"      group[ from_shared:exit_ticket（或课后迁移）, answer_box ]
```
学生页**不出现**"分层/学困/培优"等字眼；支架（如句子框架、关键词提示）作为任务自然的一部分出现，
不标"这是给你的拐杖"。

## 六、评价与出口小测

- 出口小测是教学过程的最后一环（`from_shared:exit_ticket` 置于环节标题下）。一题，指向课标要求的最难
  语言/思维应用。散文类：语言品味或写法迁移题；诗词类：意象/情感阐释题；文言文类：断句或译句题。
- 三档归类（掌握 / 基本掌握 / 需再教）用 `cards` 块呈现于教案，便于教师分拣——
  **但学生页上不写档位标签**（隐性分层，契合"不给学生贴标签"的课标评价精神）。
- 严禁以单一分数给学生定性；作业总量从严控制。
